## 🔧 性能优化技巧

### 1. 算法优化

```dart
// 算法优化
class AlgorithmOptimization {
  void explain() {
    print('''
    算法优化：
    
    // 1. 选择合适的算法
    // - 时间复杂度：O(1) < O(log n) < O(n) < O(n log n) < O(n²)
    // - 空间复杂度：选择内存占用小的算法
    
    // 2. 避免重复计算
    // 使用缓存
    final cache = <int, int>{};
    
    int fibonacci(int n) {
      if (n <= 1) return n;
      if (cache.containsKey(n)) return cache[n]!;
      
      cache[n] = fibonacci(n - 1) + fibonacci(n - 2);
      return cache[n]!;
    }
    
    // 3. 使用高效的数据结构
    // - List：适合顺序访问
    // - Set：适合查找和去重
    // - Map：适合键值对存储
    
    // 4. 避免不必要的操作
    // - 减少循环嵌套
    // - 提前退出循环
    // - 使用位运算
    // - 批量处理数据
    ''');
  }
}
```
