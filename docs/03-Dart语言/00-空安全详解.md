# 空安全详解

> 深入理解 Dart 中的空安全机制。

## 📖 空安全基础

### 1. 什么是空安全

```dart
// 空安全概念
class NullSafetyConcept {
  void explain() {
    print('''
    空安全概念：
    
    // 1. 什么是空安全
    // - 编译时防止空指针异常
    // - 类型系统区分可空和非空
    // - 显式处理空值
    // - 提高代码安全性
    
    // 2. 空安全的特点
    // - 非空默认：变量默认不可空
    // - 可空声明：使用 ? 声明可空类型
    // - 空检查：编译时检查空值
    // - 安全访问：使用 ?. 安全访问
    
    // 3. 空安全的优势
    // - 防止空指针异常
    // - 提高代码质量
    // - 改善开发体验
    // - 优化运行时性能
    
    // 4. 空安全的使用场景
    // - 变量声明：指定是否可空
    // - 函数参数：指定参数是否可空
    // - 返回类型：指定返回值是否可空
    // - 类成员：指定成员是否可空
    ''');
  }
}
```

### 2. 空安全示例

```dart
// 空安全示例
class NullSafetyExample {
  void explain() {
    print('''
    空安全示例：
    
    // 1. 非空类型
    String name = 'John';  // 非空，不能赋值为 null
    // name = null;  // 错误：不能将 null 赋值给非空类型
    
    // 2. 可空类型
    String? nullableName = 'John';  // 可空，可以赋值为 null
    nullableName = null;  // 正确：可以赋值为 null
    
    // 3. 空检查
    void process(String? text) {
      if (text != null) {
        print(text.length);  // 安全：已检查非空
      }
    }
    
    // 4. 安全访问
    String? name = null;
    int? length = name?.length;  // 安全：如果 name 为 null，返回 null
    
    // 5. 空合并运算符
    String? name = null;
    String displayName = name ?? 'Unknown';  // 如果 name 为 null，使用默认值
    
    // 6. 空断言
    String? name = 'John';
    String nonNullName = name!;  // 断言 name 不为 null
    
    // 7. late 关键字
    late String description;  // 延迟初始化
    
    void init() {
      description = 'Hello World';  // 在使用前必须初始化
    }
    ''');
  }
}
```

## 🔧 空安全机制

### 1. 可空性

```dart
// 可空性
class Nullability {
  void explain() {
    print('''
    可空性：
    
    // 1. 非空类型
    String name = 'John';  // 非空
    int age = 30;          // 非空
    bool isValid = true;   // 非空
    
    // 2. 可空类型
    String? nullableName = null;  // 可空
    int? nullableAge = null;      // 可空
    bool? nullableFlag = null;    // 可空
    
    // 3. 集合可空性
    List<int>? nullableList = null;      // 列表本身可空
    List<int?> listWithNullable = [1, null, 3];  // 元素可空
    List<int?>? nullableListWithNullable = null;  // 都可空
    
    // 4. 函数可空性
    String? getName() => null;  // 返回值可空
    void process(String? text) {}  // 参数可空
    
    // 5. 类成员可空性
    class Person {
      String name;      // 非空
      String? nickname; // 可空
      
      Person(this.name, this.nickname);
    }
    ''');
  }
}
```

### 2. 空安全操作

```dart
// 空安全操作
class NullSafetyOperators {
  void explain() {
    print('''
    空安全操作：
    
    // 1. 安全访问操作符 ?.
    String? name = null;
    int? length = name?.length;  // 如果 name 为 null，返回 null
    
    // 2. 空合并操作符 ??
    String? name = null;
    String displayName = name ?? 'Unknown';  // 如果 name 为 null，使用默认值
    
    // 3. 空合并赋值操作符 ??=
    String? name;
    name ??= 'Default';  // 如果 name 为 null，赋值为 'Default'
    
    // 4. 空断言操作符 !
    String? name = 'John';
    String nonNullName = name!;  // 断言 name 不为 null
    
    // 5. 条件属性访问
    String? name = null;
    int? length = name?.length;  // 安全访问
    
    // 6. 条件方法调用
    String? name = null;
    String? upper = name?.toUpperCase();  // 安全调用
    
    // 7. 空安全最佳实践
    // - 尽量使用非空类型
    // - 使用 ?. 安全访问
    // - 使用 ?? 提供默认值
    // - 使用 ! 时确保非空
    ''');
  }
}
```

## ⚠️ 注意事项

1. **理解空安全的概念**
2. **合理使用可空类型**
3. **注意空安全操作符**
4. **避免不必要的空断言**
