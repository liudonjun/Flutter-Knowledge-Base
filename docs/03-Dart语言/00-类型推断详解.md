# 类型推断详解

> 深入理解 Dart 中的类型推断机制。

## 📖 类型推断基础

### 1. 什么是类型推断

```dart
// 类型推断概念
class TypeInferenceConcept {
  void explain() {
    print('''
    类型推断概念：
    
    // 1. 什么是类型推断
    // - 编译器自动推断变量类型
    // - 无需显式声明类型
    // - 基于赋值和上下文推断
    // - 静态类型检查
    
    // 2. 类型推断的特点
    // - 简化代码：减少类型声明
    // - 类型安全：编译时检查
    // - 灵活性：支持动态类型
    // - 可读性：代码更简洁
    
    // 3. 类型推断的使用场景
    // - 局部变量：自动推断类型
    // - 泛型：自动推断泛型类型
    // - 函数返回：自动推断返回类型
    // - 集合：自动推断元素类型
    
    // 4. 类型推断的优势
    // - 减少代码量：无需重复声明类型
    // - 提高可读性：代码更简洁
    // - 保持安全：编译时类型检查
    // - 支持重构：类型安全重构
    ''');
  }
}
```

### 2. 类型推断示例

```dart
// 类型推断示例
class TypeInferenceExample {
  void explain() {
    print('''
    类型推断示例：
    
    // 1. 局部变量类型推断
    var name = 'John';  // 推断为 String
    var age = 30;       // 推断为 int
    var price = 19.99;  // 推断为 double
    var isValid = true; // 推断为 bool
    
    // 2. 集合类型推断
    var numbers = [1, 2, 3, 4, 5];  // 推断为 List<int>
    var names = ['John', 'Jane'];   // 推断为 List<String>
    var scores = {'math': 90, 'english': 85};  // 推断为 Map<String, int>
    
    // 3. 泛型类型推断
    var list = <int>[];  // 显式指定类型
    var map = <String, int>{};  // 显式指定类型
    
    // 4. 函数返回类型推断
    add(int a, int b) => a + b;  // 推断返回类型为 int
    
    // 5. 使用 final 和 const
    final name = 'John';  // 推断为 String，不可重新赋值
    const pi = 3.14159;   // 推断为 double，编译时常量
    
    // 6. 类型推断最佳实践
    // - 使用 var 简化局部变量声明
    // - 使用 final 保护不可变变量
    // - 使用 const 定义常量
    // - 必要时显式指定类型
    ''');
  }
}
```

## 🔧 类型推断机制

### 1. 推断规则

```dart
// 推断规则
class InferenceRules {
  void explain() {
    print('''
    推断规则：
    
    // 1. 字面量推断
    var integer = 42;        // int
    var decimal = 3.14;      // double
    var text = 'Hello';      // String
    var flag = true;         // bool
    var list = [1, 2, 3];    // List<int>
    var map = {'a': 1};      // Map<String, int>
    
    // 2. 函数推断
    int add(int a, int b) => a + b;
    var result = add(1, 2);  // 推断为 int
    
    // 3. 构造函数推断
    var list = List<int>.filled(10, 0);  // List<int>
    var map = Map<String, int>();        // Map<String, int>
    
    // 4. 泛型推断
    T first<T>(List<T> list) => list.first;
    var number = first([1, 2, 3]);  // 推断为 int
    
    // 5. 上下文推断
    List<int> numbers = [1, 2, 3];  // 上下文指定类型
    var doubled = numbers.map((n) => n * 2);  // 推断为 Iterable<int>
    
    // 6. 推断失败
    // var x = null;  // 推断为 Null
    // var list = [];  // 推断为 List<dynamic>
    // var map = {};   // 推断为 Map<dynamic, dynamic>
    ''');
  }
}
```

### 2. 类型提升

```dart
// 类型提升
class TypePromotion {
  void explain() {
    print('''
    类型提升：
    
    // 1. 条件类型提升
    void process(Object obj) {
      if (obj is String) {
        // obj 被提升为 String
        print(obj.length);  // 可以调用 String 的方法
      }
    }
    
    // 2. 空检查类型提升
    void processNullable(String? text) {
      if (text != null) {
        // text 被提升为 String
        print(text.length);  // 不需要空检查
      }
    }
    
    // 3. 逻辑与类型提升
    void processWithAnd(Object obj) {
      if (obj is String && obj.length > 0) {
        // obj 被提升为 String
        print(obj.toUpperCase());
      }
    }
    
    // 4. 类型提升限制
    void processWithLimit(Object obj) {
      if (obj is String) {
        print(obj.length);  // 可以
        // obj = 42;  // 错误：不能重新赋值为不同类型
      }
    }
    
    // 5. 类型提升最佳实践
    // - 使用 is 检查类型
    // - 使用空检查提升可空类型
    // - 使用逻辑与组合条件
    // - 注意类型提升的作用域
    ''');
  }
}
```
