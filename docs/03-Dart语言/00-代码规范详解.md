# 代码规范详解

> 深入理解 Dart 的代码规范和最佳实践。

## 📖 代码规范基础

### 1. 命名规范

```dart
// 命名规范
class NamingConventions {
  void explain() {
    print('''
    命名规范：
    
    // 1. 类名：大驼峰 (PascalCase)
    class MyClass {}
    class StatefulWidget {}
    class TextEditingController {}
    
    // 2. 变量和函数：小驼峰 (camelCase)
    int myVariable = 0;
    void myFunction() {}
    String getFullName() {}
    
    // 3. 常量：小驼峰或下划线
    const myConstant = 42;
    const MAX_VALUE = 100;
    const defaultTimeout = 30;
    
    // 4. 私有成员：下划线前缀
    class MyClass {
      int _privateVariable = 0;
      void _privateMethod() {}
    }
    
    // 5. 文件名：下划线分隔
    // my_widget.dart
    // user_model.dart
    // api_service.dart
    
    // 6. 包名：小写，下划线分隔
    // my_package
    // flutter_bloc
    // http_client
    ''');
  }
}
```

### 2. 代码格式

```dart
// 代码格式
class CodeFormatting {
  void explain() {
    print('''
    代码格式：
    
    // 1. 缩进：2 个空格
    void myFunction() {
      if (condition) {
        doSomething();
      }
    }
    
    // 2. 大括号：K&R 风格
    if (condition) {
      doSomething();
    } else {
      doSomethingElse();
    }
    
    // 3. 行长度：80 字符以内
    // 长行需要换行
    final veryLongVariableName = someFunction(
      parameter1,
      parameter2,
      parameter3,
    );
    
    // 4. 空行：逻辑分组
    class MyClass {
      // 变量声明
      int _count = 0;
      String _name = '';
      
      // 构造函数
      MyClass(this._count, this._name);
      
      // 公共方法
      void increment() {
        _count++;
      }
      
      // 私有方法
      void _privateMethod() {}
    }
    
    // 5. 空格：运算符周围
    int result = a + b * c;
    bool isValid = x > 0 && y < 100;
    String name = 'John' + ' ' + 'Doe';
    ''');
  }
}
```

## 🔧 最佳实践

### 1. 代码组织

```dart
// 代码组织
class CodeOrganization {
  void explain() {
    print('''
    代码组织：
    
    // 1. 类成员顺序
    class MyClass {
      // 1. 静态变量
      static const MAX_VALUE = 100;
      
      // 2. 实例变量
      int _count = 0;
      String _name = '';
      
      // 3. 构造函数
      MyClass(this._count, this._name);
      
      // 4. 公共方法
      void increment() {
        _count++;
      }
      
      // 5. 私有方法
      void _privateMethod() {}
      
      // 6. 重写方法
      @override
      String toString() => 'MyClass($_count, $_name)';
    }
    
    // 2. 文件组织
    // imports
    // part/part of
    // 类定义
    // 顶级函数
    // 常量
    
    // 3. 导入顺序
    // dart: 库
    // package: 库
    // 相对路径
    ''');
  }
}
```

### 2. 注释规范

```dart
// 注释规范
class CommentConventions {
  void explain() {
    print('''
    注释规范：
    
    // 1. 文档注释
    /// 这是一个文档注释
    /// 用于描述类、方法、变量
    class MyClass {
      /// 计数器的值
      int count = 0;
      
      /// 增加计数器
      /// 
      /// [amount] 要增加的数量
      void increment(int amount) {
        count += amount;
      }
    }
    
    // 2. 块注释
    /* 这是一个块注释
       用于解释复杂逻辑 */
    
    // 3. 行注释
    // 这是一个行注释
    int x = 5; // 行末注释
    
    // 4. 注释最佳实践
    // - 解释为什么，而不是是什么
    // - 保持注释更新
    // - 避免过度注释
    // - 使用有意义的注释
    ''');
  }
}
```

## ⚠️ 注意事项

1. **遵循命名规范**
2. **保持代码格式一致**
3. **合理组织代码结构**
4. **编写有意义的注释**
