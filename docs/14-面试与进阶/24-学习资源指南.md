# 学习资源指南

> Flutter / Dart 官方文档、书籍、视频、博客与开源项目学习路径。

## 官方文档

| 资源 | 链接用途 |
| --- | --- |
| [flutter.dev](https://flutter.dev) | Widget 目录、布局教程、部署 |
| [dart.dev](https://dart.dev) | 语言规范、空安全、async |
| [api.flutter.dev](https://api.flutter.dev) | API 查阅 |
| [pub.dev](https://pub.dev) | 包选型与版本 |

**建议**：以「任务驱动」阅读——做列表就查 `ListView`，做路由就查 `go_router` 文档。

## 开源项目学习

**入门读代码**：

1. [flutter/samples](https://github.com/flutter/samples) — 官方示例
2. [flutter/gallery](https://github.com/flutter/gallery) — Material 3 示范
3. 选一个中等 star 的 app  clone，从 `main.dart` → 路由 → 状态管理读起

**参与贡献**：

- 从文档 typo、测试、reproduce issue 开始
- 阅读 CONTRIBUTING.md，小 PR 积累 [[开源贡献]] 经历（面试可讲）
