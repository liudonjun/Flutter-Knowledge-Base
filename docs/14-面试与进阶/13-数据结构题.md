# 数据结构题

> 面试常见数据结构、复杂度分析与 Dart 实现要点。

## 复杂度分析

| 符号 | 含义 |
| --- | --- |
| O(1) | 常数 |
| O(log n) | 二分、平衡树 |
| O(n) | 线性扫描 |
| O(n log n) | 排序 |
| O(n²) | 双重循环 |

**空间复杂度**：辅助数组/哈希表计入。

**题：HashMap 查找平均 O(1) 原因？** 良好哈希 + 扩容；最坏 O(n) 链表/树。

---

## 数组与动态数组

Dart `List` 底层动态数组：

- 末尾 `add` 均摊 O(1)
- 中间 `insert` O(n)
- `contains` O(n)；用 `Set` O(1)

**题：删除有序数组重复项（原地）** — 双指针 slow/fast。

---

## 链表

| 操作 | 时间 |
| --- | --- |
| 访问 i | O(n) |
| 头插 | O(1) |
| 删已知节点 | O(1) 若给指针 |

**题：环检测** — Floyd 快慢指针。

**题：合并两个有序链表** — 哑节点 dummy head。

## 栈与队列

```dart
import 'dart:collection';

final stack = <int>[];
stack.addLast(1);
stack.removeLast();

final queue = Queue<int>();
queue.addLast(1);
queue.removeFirst();
```

**题：有效括号** — 栈匹配。

**题：BFS 最短路径** — 队列 + visited。

---

## 哈希表

Dart：`Map`、`Set`、`LinkedHashMap`（默认有序键插入序）。

**题：两数之和、字母异位词、最长无重复子串** — 滑动窗口 + Map。

**题：LRU 缓存** — `LinkedHashMap` 或 双向链表 + Map（中等难度）。

---

## 二叉树

```dart
class TreeNode {
  TreeNode(this.val, [this.left, this.right]);
  int val;
  TreeNode? left, TreeNode? right;
}
```

| 遍历 | 顺序 | 常用 |
| --- | --- | --- |
| 前序 | 根左右 | 复制树 |
| 中序 | 左根右 | BST 有序 |
| 后序 | 左右根 | 删树 |
| 层序 | BFS | 最短层 |

**题：最大深度** — `1 + max(left, right)`。

**题：验证 BST** — 中序递增或传 min/max 边界。

**题：最近公共祖先** — 递归，左右都非空则当前为 LCA。

---

## 堆（优先级队列）

```dart
import 'dart:collection';
final heap = HeapPriorityQueue<int>((a, b) => a.compareTo(b));
```

**题：Top K 频繁元素** — 小顶堆 size K，O(n log k)。
