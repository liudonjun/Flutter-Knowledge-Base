# 编译原理与引擎

> Dart 编译链路、类型系统概览与 Skia / Impeller 渲染引擎入门。

## JIT / AOT / Web

| 模式 | 场景 | 特点 |
| --- | --- | --- |
| **JIT** | `flutter run` debug | 热重载、启动较慢 |
| **AOT** | release APK/IPA | 机器码、体积小、快 |
| **dart2js / wasm** | Web | 不同后端，关注包体积 |

**[[编译原理]]** 面试答法：Dart 源码 → Kernel →（JIT 或 AOT）→ 原生或 VM 执行；release 无反射式热更。

## 类型系统

- 静态类型 +  sound null safety。
- 泛型、函数类型、记录（Records）、sealed class。
- 与 Java/C# 对比：无 primitive 装箱重复、everything is object（除 null 安全区分）。

深入：[[19-Dart深入]]、[[Dart源码]] 阅读 `sdk/pkg/analyzer`。

## Skia vs Impeller

| | Skia | Impeller |
| --- | --- | --- |
| **性质** | 成熟 2D 引擎 | Flutter 自研，预编译 shader |
| **问题** | 首帧 shader jank | 降低 GPU 编译卡顿 |
| **平台** | 全平台 | iOS 默认，Android 推进中 |

**[[Skia深入]]**：Layer 树、Picture → SkCanvas 绘制；Flutter `RenderObject.paint`。

**[[图形学]]** 基础：矢量 vs 位图、抗锯齿、变换矩阵（了解即可）。
