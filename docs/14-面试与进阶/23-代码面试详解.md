# 代码面试详解

> 算法题与 Flutter 手写题的 preparation、答题流程与常见题型。

## 复杂度分析

答题前先说明 **时间 / 空间复杂度**：

| 符号 | 含义 | 典型算法 |
| --- | --- | --- |
| O(1) | 常数 | 哈希查找（平均） |
| O(log n) | 对数 | 二分、平衡树 |
| O(n) | 线性 | 单次遍历 |
| O(n log n) | 线性对数 | 排序 |
| O(n²) | 平方 | 双重循环 |

口述模板：「暴力 O(n²)，用哈希表可降到 O(n) 时间、O(n) 额外空间。」

## 常见题型

1. **数组 / 字符串**：双指针、滑动窗口、前缀和。
2. **链表**：快慢指针、反转、环检测。
3. **树 / 图**：DFS、BFS、层序遍历。
4. **动态规划**：状态定义 + 转移（先 brute force 再优化）。

Flutter/Dart 面试也可能考 **Dart 语言题**：Future 链、Stream、闭包捕获。

详见 [[11-Flutter算法题]]、[[13-数据结构题]]。

## Flutter 手写 Widget

高频题：

| 题目 | 要点 |
| --- | --- |
| **计数器** | StatefulWidget、`setState` |
| **无限列表** | `ListView.builder`、分页 |
| **搜索框 + 防抖** | `TextEditingController`、`Timer` |
| **主题切换** | `InheritedWidget` / Provider |

```dart
class DebouncedSearch extends StatefulWidget {
  const DebouncedSearch({super.key});

  @override
  State<DebouncedSearch> createState() => _DebouncedSearchState();
}

class _DebouncedSearchState extends State<DebouncedSearch> {
  Timer? _debounce;
  String _query = '';

  void _onChanged(String value) {
    _debounce?.cancel();
    _debounce = Timer(const Duration(milliseconds: 300), () {
      setState(() => _query = value);
      // trigger search with _query
    });
  }

  @override
  void dispose() {
    _debounce?.cancel();
    super.dispose();
  }

  @override
  Widget build(BuildContext context) {
    return TextField(onChanged: _onChanged);
  }
}
```
