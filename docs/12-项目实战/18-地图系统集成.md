# 地图系统集成

> Flutter 接入地图 SDK、定位与路径规划的实战要点。

## 常见方案

| 包 / 方案 | 说明 |
| --- | --- |
| **google_maps_flutter** | Google 地图，国内需考虑可用性 |
| **amap_flutter_map** 等 | 国内高德等原生 SDK 插件 |
| **mapbox_maps_flutter** | Mapbox 定制样式 |

选型：目标市场、离线需求、费用、插件维护活跃度。

---

## 集成步骤概览

1. 各平台申请 API Key，配置 `AndroidManifest` / `Info.plist`。
2. `permission_handler` + [[58-geolocator详解]] 申请定位权限。
3. 地图 Widget 嵌入页面，处理生命周期（暂停/恢复地图）。
4. Marker、Polyline、相机移动等平台 API 封装为 Repository。

---

## 性能与体验

- 列表页 **不要用** 多个实时地图 Tile（极耗 GPU/内存）。
- 静态缩略图 + 详情页全屏地图。
- 大量 Marker 用聚合（cluster）或服务端过滤可见区域。
