# 社交应用实战

> 用户关系、动态 Feed、即时通讯与推送的 Flutter 社交项目架构、模块实现与迭代计划。

## 项目目标

| 模块 | 功能 |
| --- | --- |
| **用户** | 注册登录、资料、关注/粉丝 — [[12-用户系统设计]] |
| **动态** | 发布、Feed 流、点赞评论、分享 |
| **消息** | 单聊/群聊、未读数 — [[13-消息系统设计]] |
| **发现** | 搜索、推荐好友 |
| **推送** | FCM / 本地通知 — [[24-firebase_messaging详解]]、[[25-flutter_local_notifications详解]] |

## 推荐技术栈

- **状态管理**：Riverpod / Bloc（会话列表、Feed 分页、Tab 状态）
- **REST**：Dio 拉 Feed、资料、关系链 — [[03-网络请求库详解]]
- **实时**：WebSocket / [[41-socket_io_client详解]] 或 Firebase Realtime
- **媒体**：[[22-image_picker详解]] + 对象存储（S3/OSS）
- **第三方登录**（可选）：[[38-wechat_login详解]]
- **推送**：FCM Token 绑定 userId

### 即时通讯

```dart
enum MessageType { text, image, system }

class Message {
  Message({
    required this.id,
    required this.senderId,
    required this.receiverId,
    required this.content,
    required this.type,
    required this.timestamp,
    this.isRead = false,
  });

  final String id;
  final String senderId;
  final String receiverId;
  final String content;
  final MessageType type;
  final DateTime timestamp;
  final bool isRead;

  factory Message.fromJson(Map<String, dynamic> json) => Message(
        id: json['id'] as String,
        senderId: json['sender_id'] as String,
        receiverId: json['receiver_id'] as String,
        content: json['content'] as String,
        type: MessageType.values.byName(json['type'] as String),
        timestamp: DateTime.parse(json['timestamp'] as String),
        isRead: json['is_read'] as bool? ?? false,
      );
}

class ChatService {
  ChatService(this._channelFactory);
  final WebSocketChannel Function(Uri) _channelFactory;
  late WebSocketChannel _channel;
  final _controller = StreamController<Message>.broadcast();

  Stream<Message> get messages => _controller.stream;

  void connect(String url, {required String token}) {
    _channel = _channelFactory(Uri.parse('$url?token=$token'));
    _channel.stream.listen((data) {
      _controller.add(Message.fromJson(jsonDecode(data as String)));
    });
  }

  void send(Message message) {
    _channel.sink.add(jsonEncode({
      'sender_id': message.senderId,
      'receiver_id': message.receiverId,
      'content': message.content,
      'type': message.type.name,
    }));
  }

  Future<void> disconnect() async {
    await _channel.sink.close();
    await _controller.close();
  }
}
```

**消息状态机**：

```
连接 WS → auth(token) → 订阅会话
收到 message → 写本地 DB → 更新未读 → 通知 UI
发送 → 本地 pending → ack 后改 sent
```

- 前台：socket 直推；后台/进程被杀：FCM 唤醒或拉取增量
- 聊天页：`ListView.builder` **`reverse: true`**，历史消息向上分页加载
- 心跳与指数退避重连，避免弱网风暴

### 网络层

```dart
class ApiClient {
  ApiClient({required String baseUrl})
      : _dio = Dio(BaseOptions(
          baseUrl: baseUrl,
          connectTimeout: const Duration(seconds: 5),
          receiveTimeout: const Duration(seconds: 10),
        )) {
    _dio.interceptors.add(AuthInterceptor());
  }

  final Dio _dio;

  Future<Response<T>> get<T>(String path, {Map<String, dynamic>? query}) =>
      _dio.get(path, queryParameters: query);

  Future<Response<T>> post<T>(String path, {dynamic data}) =>
      _dio.post(path, data: data);
}

class AuthInterceptor extends Interceptor {
  @override
  void onRequest(RequestOptions options, RequestInterceptorHandler handler) {
    final token = TokenStore.accessToken;
    if (token != null) {
      options.headers['Authorization'] = 'Bearer $token';
    }
    handler.next(options);
  }
}
```

### 图片上传

```dart
class ImageUploadService {
  ImageUploadService(this._dio);
  final Dio _dio;

  Future<String> upload(File file) async {
    final form = FormData.fromMap({
      'file': await MultipartFile.fromFile(file.path, filename: 'image.jpg'),
    });
    final res = await _dio.post('/upload/image', data: form);
    return res.data['url'] as String;
  }
}
```

上传前压缩；多图串行或限并发，避免 OOM。
