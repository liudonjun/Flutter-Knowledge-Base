# 电商应用实战

> 从商品浏览到下单支付的 Flutter 电商项目架构、模块实现与迭代计划。

## 项目目标

| 模块 | 功能 |
| --- | --- |
| **商品** | 分类、列表、详情、搜索 |
| **购物车** | 加购、改数量、失效商品 |
| **订单** | 下单、状态流转、历史订单 |
| **用户** | 登录注册、地址、个人中心 — [[12-用户系统设计]] |
| **支付** | 多渠道支付 — [[11-支付系统设计]] |

## 推荐技术栈

- **状态管理**：Riverpod 或 Bloc（购物车、登录态全局）
- **网络**：Dio + [[08-retrofit详解]]
- **本地**：Hive / SQLite 缓存购物车草稿 — [[09-shared_preferences详解]]、[[04-数据库库详解]]
- **支付**：国内 [[33-alipay详解]]、[[34-wechat_pay详解]]；国际 [[32-stripe详解]]
- **图片**：`cached_network_image`，指定 `memCacheWidth`

## 功能模块与代码要点

### 商品模块

```dart
class Product {
  const Product({
    required this.id,
    required this.name,
    required this.price,
    required this.imageUrl,
    this.description = '',
  });

  final String id;
  final String name;
  final double price;
  final String imageUrl;
  final String description;

  factory Product.fromJson(Map<String, dynamic> json) => Product(
        id: json['id'] as String,
        name: json['name'] as String,
        price: (json['price'] as num).toDouble(),
        imageUrl: json['image_url'] as String,
        description: json['description'] as String? ?? '',
      );
}
```

- 列表：`GridView.builder` + 分页；cell 固定宽高
- 详情：轮播图 + SKU 选择（有规格时状态用独立 `SkuSelectionNotifier`）
- 搜索：防抖 300ms，见 [[23-代码面试详解]]

### 购物车

```dart
class CartItem {
  CartItem({
    required this.productId,
    required this.qty,
    required this.unitPrice,
    this.title,
  });

  final String productId;
  int qty;
  final double unitPrice;
  final String? title;

  double get lineTotal => qty * unitPrice;
}

class CartNotifier extends Notifier<List<CartItem>> {
  @override
  List<CartItem> build() => [];

  void add(Product p, {int qty = 1}) {
    final i = state.indexWhere((e) => e.productId == p.id);
    if (i >= 0) {
      state = [
        for (var j = 0; j < state.length; j++)
          if (j == i) CartItem(productId: p.id, qty: state[j].qty + qty, unitPrice: p.price, title: p.name)
          else state[j],
      ];
    } else {
      state = [...state, CartItem(productId: p.id, qty: qty, unitPrice: p.price, title: p.name)];
    }
  }
}
```

- 未登录：本地持久化；登录后与服务端 cart merge
- 结算前：**重新拉价格与库存**，防止过期缓存

### 用户与 API

```dart
class OrderApi {
  OrderApi(this._dio);
  final Dio _dio;

  Future<List<Product>> getProducts({required int page}) async {
    final res = await _dio.get('/products', queryParameters: {'page': page});
    return (res.data as List).map((e) => Product.fromJson(e)).toList();
  }

  Future<String> createOrder(CreateOrderRequest req) async {
    final res = await _dio.post('/orders', data: req.toJson());
    return res.data['order_id'] as String;
  }
}
```

Token 使用 [[20-secure_storage详解]]；401 走刷新或重新登录。

### 支付链路（多渠道）

```dart
abstract class PaymentGateway {
  Future<PayResult> pay(String orderId);
}

// 注册：AlipayGateway / WechatPayGateway / StripeGateway
```

1. `POST /orders` → `order_id`
2. `POST /payments/{channel}` → 渠道参数（orderInfo / prepay / client_secret）
3. 调 SDK
4. **轮询 `GET /orders/{id}` 或等 webhook**，再展示成功页

国内渠道详见 [[33-alipay详解]]、[[34-wechat_pay详解]]。
