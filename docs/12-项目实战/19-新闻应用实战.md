# 新闻应用实战

> 新闻/资讯类 Flutter 应用的模块划分与典型技术点。

## 核心模块

| 模块 | 技术点 |
| --- | --- |
| **Feed 流** | 分页、`ListView.builder`、下拉刷新 `RefreshIndicator` |
| **详情** | `WebView` 或原生富文本、分享 [[27-share_plus详解]] |
| **分类 / Tab** | `TabBar` + `AutomaticKeepAliveClientMixin` |
| **搜索** | 防抖、历史记录 [[09-shared_preferences详解]] |
| **离线** | 缓存文章 JSON、图片缓存 |

---

## 数据流

```
API → Repository（内存 + 本地缓存）→ Bloc/Riverpod → UI
```

- 首屏：缓存优先，后台刷新（stale-while-revalidate）。
- 错误：区分无网络 / 服务器错误，可重试。

---

## 性能建议

- 列表 `itemExtent` 或固定高度 cell。
- 图片 `cacheWidth`；占位色块。
- 详情页 `RepaintBoundary` 隔离 WebView（若用）。

参考站内 [[07-系统设计面试题]] Feed 设计考点。
