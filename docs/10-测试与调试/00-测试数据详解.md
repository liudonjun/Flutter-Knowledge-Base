## 📖 测试数据基础

### 1. 测试数据概念

```dart
// 测试数据概念
class TestDataConcept {
  void explain() {
    print('''
    测试数据概念：
    
    // 1. 什么是测试数据
    // - 测试使用的数据
    // - 模拟真实数据
    // - 支持测试场景
    // - 提高测试效率
    
    // 2. 测试数据类型
    // - 固定数据：硬编码数据
    // - 随机数据：随机生成
    // - 工厂数据：工厂模式
    // - Fixture：测试夹具
    
    // 3. 测试数据管理
    // - 数据组织
    // - 数据复用
    // - 数据清理
    // - 数据版本
    
    // 4. 测试数据工具
```

### 2. 测试数据实现

```dart
// 测试数据实现
class TestDataImplementation {
  void explain() {
    print('''
    测试数据实现：
    
    // 1. 固定数据
    final testData = {
      'name': 'John',
      'age': 30,
      'email': 'john@example.com',
    };
    
    // 2. 工厂模式
    class UserFactory {
      static User create({
        String name = 'John',
        int age = 30,
      }) {
        return User(name: name, age: age);
      }
    }
    
    // 3. 测试夹具
    class UserFixture {
      static User get user => User(name: 'John', age: 30);
      static List<User> get users => [
        User(name: 'John', age: 30),
        User(name: 'Jane', age: 25),
      ];
    }
    
    // 4. 使用 faker
    import 'package:faker/faker.dart';
    
    final faker = Faker();
    final user = User(
      name: faker.person.name(),
      email: faker.internet.email(),
    );
    
    // 5. 测试数据最佳实践
    // - 使用工厂模式
    // - 保持数据简单
    // - 复用测试数据
    // - 清理测试数据
    ''');
  }
}
```
