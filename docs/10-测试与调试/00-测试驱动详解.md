### 2. 测试驱动开发实现

```dart
// 测试驱动开发实现
class TestDrivenDevelopmentImplementation {
  void explain() {
    print('''
    测试驱动开发实现：
    
    // 1. 红：写失败测试
    test('counter increments', () {
      final counter = Counter();
      counter.increment();
      expect(counter.count, 1);  // 失败
    });
    
    // 2. 绿：写代码通过
    class Counter {
      int _count = 0;
      int get count => _count;
      void increment() => _count++;
    }
    
    // 3. 重构：优化代码
    class Counter {
      int _count = 0;
      int get count => _count;
      void increment() => _count++;
      void decrement() => _count--;
      void reset() => _count = 0;
    }
    
    // 4. 测试驱动开发最佳实践
    // - 小步前进
    // - 频繁测试
    // - 保持简单
    // - 持续重构
    ''');
  }
}
```
