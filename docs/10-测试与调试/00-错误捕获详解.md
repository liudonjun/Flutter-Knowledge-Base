# 错误捕获详解

> 深入理解 Flutter 中的错误捕获。

## 📖 错误捕获基础

### 1. 错误捕获方法

```dart
// 错误捕获方法
class ErrorCatchingMethods {
  void explain() {
    print('''
    错误捕获方法：
    
    // 1. try-catch
    try {
      // 可能出错的代码
    } catch (e) {
      // 错误处理
    } finally {
      // 清理代码
    }
    
    // 2. FlutterError
    FlutterError.onError = (details) {
      // 处理 Flutter 错误
    };
    
    // 3. PlatformDispatcher
    PlatformDispatcher.instance.onError = (error, stack) {
      // 处理平台错误
      return true;
    };
    
    // 4. Zone
    runZonedGuarded(() {
      // 应用代码
    }, (error, stack) {
      // 错误处理
    });
    ''');
  }
}
```
