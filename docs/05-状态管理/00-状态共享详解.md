## 📖 状态共享基础

### 1. 什么是状态共享

```dart
// 状态共享概念
class StateSharingConcept {
  void explain() {
    print('''
    状态共享概念：

    // - 状态变化通知所有依赖
    // - 实现数据同步
    // - 避免状态重复
    
    // 2. 状态共享的特点

    // 4. 状态共享的使用场景
```

### 2. 状态共享实现

```dart
// 状态共享实现
class StateSharingImplementation {
  void explain() {
    print('''
    状态共享实现：
    
    // 1. 使用 InheritedWidget
    class SharedState extends InheritedWidget {
      final int count;
      
      const SharedState({
        Key? key,
        required this.count,
        required Widget child,
      }) : super(key: key, child: child);
      
      static SharedState? of(BuildContext context) {
        return context.dependOnInheritedWidgetOfExactType<SharedState>();
      }
      
      @override
      bool updateShouldNotify(SharedState oldWidget) {
        return count != oldWidget.count;
      }
    }
    
    // 2. 使用 Provider
    ChangeNotifierProvider(
      create: (context) => Counter(),
      child: MyApp(),
    )
    
    // 3. 使用 BLoC
    BlocProvider(
      create: (context) => CounterBloc(),
      child: MyApp(),
    )
    
    // 4. 使用 Redux
    StoreProvider(
      store: store,
      child: MyApp(),
    )
    ''');
  }
}
```
