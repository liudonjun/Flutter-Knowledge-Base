### 2. 重建性能优化

```dart
// 重建性能优化
class RebuildPerformanceOptimization {
  void explain() {
    print('''
    重建性能优化：
    
    // 1. 使用 const
    const Text('Hello')
    const Icon(Icons.star)
    
    // 2. 使用 Selector
    Selector<Counter, int>(
      selector: (context, counter) => counter.count,
      builder: (context, count, child) {
        return Text('Count: $count');
      },
    )
    
    // 3. 使用 RepaintBoundary
    RepaintBoundary(
      child: ComplexWidget(),
    )
    
    // 4. 使用 child 参数
    Consumer<Counter>(
      builder: (context, counter, child) {
        return Column(
          children: [
            child!,  // 不会重建
            Text('Count: ${counter.count}'),
          ],
        );
      },
      child: ExpensiveWidget(),
    )
    
    // 5. 重建性能最佳实践
    // - 最小化重建范围
    // - 减少重建频率
    // - 优化重建复杂度
    // - 使用性能工具
    ''');
  }
}
```
