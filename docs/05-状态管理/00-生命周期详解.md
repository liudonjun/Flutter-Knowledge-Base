# 生命周期详解

> 深入理解 Flutter 中的生命周期。

## 📖 生命周期基础

### 1. 什么是生命周期

```dart
// 生命周期概念
class LifecycleConcept {
  void explain() {
    print('''
    生命周期概念：
    
    // 1. 什么是生命周期
    // - Widget 从创建到销毁的过程
    // - 状态从初始化到销毁的过程
    // - 应用从启动到关闭的过程
    // - 资源从分配到释放的过程
    
    // 2. 生命周期的特点
    // - 阶段性：有多个阶段
    // - 有序性：阶段有序执行
    // - 可控性：可以控制生命周期
    // - 资源管理：管理资源分配
    
    // 3. 生命周期的阶段
    // - 初始化：initState
    // - 构建：build
    // - 更新：didUpdateWidget
    // - 销毁：dispose
    
    // 4. 生命周期的使用场景
    // - 资源管理
    // - 状态初始化
    // - 状态清理
    // - 生命周期控制
    ''');
  }
}
```

### 2. 生命周期实现

```dart
// 生命周期实现
class LifecycleImplementation {
  void explain() {
    print('''
    生命周期实现：
    
    // 1. StatefulWidget 生命周期
    class MyWidget extends StatefulWidget {
      @override
      _MyWidgetState createState() => _MyWidgetState();
    }
    
    class _MyWidgetState extends State<MyWidget> {
      @override
      void initState() {
        super.initState();
        // 初始化
      }
      
      @override
      void didChangeDependencies() {
        super.didChangeDependencies();
        // 依赖改变
      }
      
      @override
      Widget build(BuildContext context) {
        return Container();
      }
      
      @override
      void didUpdateWidget(MyWidget oldWidget) {
        super.didUpdateWidget(oldWidget);
        // Widget 更新
      }
      
      @override
      void dispose() {
        // 清理资源
        super.dispose();
      }
    }
    
    // 2. ChangeNotifier 生命周期
    class MyNotifier extends ChangeNotifier {
      MyNotifier() {
        // 初始化
      }
      
      @override
      void dispose() {
        // 清理资源
        super.dispose();
      }
    }
    ''');
  }
}
```
