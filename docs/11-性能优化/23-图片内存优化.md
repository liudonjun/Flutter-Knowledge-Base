### 1. 使用适当尺寸

```dart
// 使用适当尺寸
class UseAppropriateSize {
  void explain() {
    print('''
    使用适当尺寸：
    
    // 1. 为什么使用适当尺寸
    // - 减少内存占用
    // - 提高性能
    // - 改善用户体验
    // - 减少网络请求
    
    // 2. 使用适当尺寸示例
    // 好
    Image.network(
      'https://example.com/image.jpg',
      width: 100,
      height: 100,
      cacheWidth: 100,   // 缓存尺寸
      cacheHeight: 100,  // 缓存尺寸
    )
    
    // 不好
    Image.network(
      'https://example.com/image.jpg',  // 原始尺寸，可能很大
    )
    
    // 3. 使用适当尺寸最佳实践
    // - 根据显示尺寸设置
    // - 使用 cacheWidth 和 cacheHeight
    // - 使用缩略图
    // - 使用响应式尺寸
    
    // 4. 使用适当尺寸注意事项
    // - 尺寸要适当
    // - 考虑显示设备
    // - 考虑网络带宽
    // - 考虑性能影响
    ''');
  }
}
```
