### 2. 布局性能分析

```dart
// 布局性能分析
class LayoutPerformanceAnalysis {
  void explain() {
    print('''
    布局性能分析：
    
    // 1. 使用 Performance Overlay
    MaterialApp(
      showPerformanceOverlay: true,
      home: MyHomePage(),
    )
    
    // 2. 使用 DevTools
    // - 打开 DevTools
    // - 切换到 Performance 面板
    // - 查看帧图表
    // - 分析布局性能
    
    // 3. 使用 Flutter Inspector
    // - 查看 Widget 树
    // - 分析布局结构
    // - 识别性能问题
    // - 优化布局
    
    // 4. 自定义性能分析
    class LayoutPerformanceTracker {
      final List<LayoutTiming> _timings = [];
      
      void startTracking() {
        SchedulerBinding.instance.addTimingsCallback((timings) {
          for (final timing in timings) {
            _timings.add(LayoutTiming(
              buildDuration: timing.buildDuration,
              rasterDuration: timing.rasterDuration,
            ));
          }
        });
      }
      
      void printReport() {
        print('布局性能报告:');
        for (final timing in _timings) {
          print('  build: ${timing.buildDuration.inMilliseconds}ms');
          print('  raster: ${timing.rasterDuration.inMilliseconds}ms');
        }
      }
    }
    ''');
  }
}
```

## 🔧 布局优化

### 1. 减少嵌套层级

```dart
// 减少嵌套层级
class ReduceNesting {
  void explain() {
    print('''
    减少嵌套层级：
    
    // 1. 为什么减少嵌套
    // - 减少布局计算
    // - 提高性能
    // - 代码更简洁
    // - 便于维护
    
    // 2. 减少嵌套的方法
    // - 使用 Padding 替代 Container
    // - 使用 SizedBox 替代 Container
    // - 使用 Align 替代 Container
    // - 使用 DecoratedBox 替代 Container
    
    // 3. 减少嵌套的示例
    // 不好
    Container(
      padding: EdgeInsets.all(8),
      child: Container(
        decoration: BoxDecoration(
          color: Colors.blue,
          borderRadius: BorderRadius.circular(8),
        ),
        child: Text('Hello'),
      ),
    )
    
    // 好
    Padding(
      padding: EdgeInsets.all(8),
      child: DecoratedBox(
        decoration: BoxDecoration(
          color: Colors.blue,
          borderRadius: BorderRadius.circular(8),
        ),
        child: Text('Hello'),
      ),
    )
    
    // 4. 减少嵌套的最佳实践
    // - 使用专用 Widget
    // - 避免不必要的 Container
    // - 使用 const 构造函数
    // - 保持布局简单
    ''');
  }
}
```

### 2. 简化布局结构

```dart
// 简化布局结构
class SimplifyLayout {
  void explain() {
    print('''
    简化布局结构：
    
    // 1. 为什么简化布局
    // - 提高性能
    // - 代码更简洁
    // - 便于维护
    // - 减少错误
    
    // 2. 简化布局的方法
    // - 使用 Row/Column 替代 Flex
    // - 使用 Wrap 替代 Row/Column
    // - 使用 Stack 替代 Positioned
    // - 使用 GridView 替代手动布局
    
    // 3. 简化布局的示例
    // 不好
    Flex(
      direction: Axis.horizontal,
      children: [
        Expanded(child: Text('Left')),
        Expanded(child: Text('Right')),
      ],
    )
    
    // 好
    Row(
      children: [
        Expanded(child: Text('Left')),
        Expanded(child: Text('Right')),
      ],
    )
    
    // 4. 简化布局的最佳实践
    // - 使用标准布局 Widget
    // - 避免复杂布局
    // - 使用布局构建器
    // - 保持布局简单
    ''');
  }
}
```
