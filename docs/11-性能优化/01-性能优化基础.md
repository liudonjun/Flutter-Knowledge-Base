# 性能优化基础

> 掌握 Flutter 性能优化的核心概念和技巧。

## 📖 性能指标

### 1. 帧率

```dart
// 帧率优化
class FrameRate {
  void explain() {
    print('''
    帧率优化：
    
    // 目标：60 FPS（每帧 16ms）
    
    // 1. 启用性能叠加
    MaterialApp(
      showPerformanceOverlay: true,
      home: MyHomePage(),
    )
    
    // 2. 检查帧率
    SchedulerBinding.instance.addTimingsCallback((timings) {
      for (final timing in timings) {
        final buildDuration = timing.buildDuration.inMilliseconds;
        final rasterDuration = timing.rasterDuration.inMilliseconds;
        
        if (buildDuration > 16 || rasterDuration > 16) {
          print('慢帧: build=${buildDuration}ms, raster=${rasterDuration}ms');
        }
      }
    });
    
    // 3. 帧率优化技巧
    // - 使用 const 构造函数
    // - 避免不必要的重建
    // - 使用 RepaintBoundary
    // - 优化图片加载
    ''');
  }
}
```

## 📖 Widget 优化

### 1. 重建优化

```dart
// 重建优化
class RebuildOptimization {
  void explain() {
    print('''
    重建优化：
    
    // 1. 使用 const
    const Text('Hello')
    const Icon(Icons.star)
    
    // 2. 提取 Widget
    class MyPage extends StatelessWidget {
      @override
      Widget build(BuildContext context) {
        return Column(
          children: [
            _buildHeader(),
            _buildContent(),
            _buildFooter(),
          ],
        );
      }
      
      Widget _buildHeader() => Text('Header');
      Widget _buildContent() => Text('Content');
      Widget _buildFooter() => Text('Footer');
    }
    
    // 3. 使用 RepaintBoundary
    RepaintBoundary(
      child: ComplexWidget(),
    )
    
    // 4. 使用 child 参数
    Consumer<Controller>(
      builder: (context, controller, child) {
        return Column(children: [child!, Text('${controller.value}')]);
      },
      child: ExpensiveWidget(),  // 不会重建
    )
    ''');
  }
}
```

### 2. 布局优化

```dart
// 布局优化
class LayoutOptimization {
  void explain() {
    print('''
    布局优化：
    
    // 1. 减少嵌套
    // 不好
    Container(
      child: Padding(
        padding: EdgeInsets.all(8),
        child: Column(...),
      ),
    )
    
    // 好
    Padding(
      padding: EdgeInsets.all(8),
      child: Column(...),
    )
    
    // 2. 使用 SizedBox 替代 Container
    SizedBox(width: 100, height: 50)
    
    // 3. 使用 LayoutBuilder
    LayoutBuilder(
      builder: (context, constraints) {
        if (constraints.maxWidth > 600) {
          return WideLayout();
        } else {
          return NarrowLayout();
        }
      },
    )
    
    // 4. 使用 MediaQuery 优化
    // 不好：每次都获取
    // 好：在 build 外获取
    @override
    Widget build(BuildContext context) {
      final size = MediaQuery.of(context).size;
      return Container(...);
    }
    ''');
  }
}
```
