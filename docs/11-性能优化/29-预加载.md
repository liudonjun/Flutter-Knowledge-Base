### 2. 预加载分析

```dart
// 预加载分析
class PreloadingAnalysis {
  void explain() {
    print('''
    预加载分析：
    
    // 1. 使用 Performance Overlay
    MaterialApp(
      showPerformanceOverlay: true,
      home: MyHomePage(),
    )
    
    // 2. 使用 DevTools
    // - 打开 DevTools
    // - 切换到 Performance 面板
    // - 查看预加载帧
    // - 分析预加载流程
    
    // 3. 使用 Flutter Inspector
    // - 查看 Widget 树
    // - 分析预加载结构
    // - 识别性能问题
    // - 优化预加载流程
    
    // 4. 自定义性能分析
    class PreloadingTracker {
      final List<PreloadingTiming> _timings = [];
      
      void startTracking() {
        SchedulerBinding.instance.addTimingsCallback((timings) {
          for (final timing in timings) {
            _timings.add(PreloadingTiming(
              buildDuration: timing.buildDuration,
              rasterDuration: timing.rasterDuration,
            ));
          }
        });
      }
      
      void printReport() {
        print('预加载性能报告:');
        for (final timing in _timings) {
          print('  build: ${timing.buildDuration.inMilliseconds}ms');
          print('  raster: ${timing.rasterDuration.inMilliseconds}ms');
        }
      }
    }
    ''');
  }
}
```

## 🔧 预加载实现

### 1. 图片预加载

```dart
// 图片预加载
class ImagePreloading {
  void explain() {
    print('''
    图片预加载：
    
    // 1. 为什么图片预加载
    // - 减少等待时间
    // - 提高性能
    // - 改善用户体验
    // - 减少网络请求
    
    // 2. 图片预加载示例
    class ImagePreloader {
      final List<String> _imageUrls = [];
      final Map<String, Image> _cachedImages = {};
      
      Future<void> preloadImages() async {
        for (final url in _imageUrls) {
          final image = await _loadImage(url);
          _cachedImages[url] = image;
        }
      }
      
      Future<Image> _loadImage(String url) async {
        // 加载图片
        return Image.network(url);
      }
      
      Image? getCachedImage(String url) {
        return _cachedImages[url];
      }
    }
    
    // 3. 图片预加载最佳实践
    // - 使用缓存
    // - 使用异步加载
    // - 使用占位符
    // - 使用错误处理
    
    // 4. 图片预加载注意事项
    // - 预加载时机要适当
    // - 使用缓存
    // - 使用占位符
    // - 考虑性能影响
    ''');
  }
}
```
