### 2. 帧率监控

```dart
// 帧率监控
class FrameRateMonitor {
  void explain() {
    print('''
    帧率监控：
    
    // 1. 使用性能叠加
    MaterialApp(
      showPerformanceOverlay: true,  // 显示性能叠加
      home: MyHomePage(),
    )
    
    // 2. 监听帧率回调
    SchedulerBinding.instance.addTimingsCallback((timings) {
      for (final timing in timings) {
        final buildDuration = timing.buildDuration.inMilliseconds;
        final rasterDuration = timing.rasterDuration.inMilliseconds;
        
        if (buildDuration > 16 || rasterDuration > 16) {
          print('慢帧: build=${buildDuration}ms, raster=${rasterDuration}ms');
        }
      }
    });
    
    // 3. 使用 DevTools
    // - 打开 DevTools
    // - 切换到 Performance 面板
    // - 点击 Record 开始录制
    // - 操作应用
    // - 点击 Stop 停止录制
    // - 分析帧率图表
    
    // 4. 自定义帧率监控
    class FrameRateTracker {
      final List<int> _frameTimes = [];
      
      void startTracking() {
        SchedulerBinding.instance.addTimingsCallback(_onFrameTimings);
      }
      
      void _onFrameTimings(List<FrameTiming> timings) {
        for (final timing in timings) {
          _frameTimes.add(timing.totalSpan.inMilliseconds);
        }
      }
      
      double get averageFrameTime {
        if (_frameTimes.isEmpty) return 0;
        return _frameTimes.reduce((a, b) => a + b) / _frameTimes.length;
      }
      
      double get averageFps {
        if (averageFrameTime == 0) return 0;
        return 1000 / averageFrameTime;
      }
    }
    ''');
  }
}
```

## 🔧 帧率优化

### 1. 减少重建

```dart
// 减少重建
class ReduceRebuilds {
  void explain() {
    print('''
    减少重建：
    
    // 1. 使用 const 构造函数
    const Text('Hello World')
    const Icon(Icons.star)
    const SizedBox(width: 100, height: 50)
    
    // 2. 提取不变部分
    class MyWidget extends StatelessWidget {
      @override
      Widget build(BuildContext context) {
        return Column(
          children: [
            _buildHeader(),      // 提取方法
            _buildContent(),
            _buildFooter(),
          ],
        );
      }
      
      Widget _buildHeader() => const Text('Header');
      Widget _buildContent() => const Text('Content');
      Widget _buildFooter() => const Text('Footer');
    }
    
    // 3. 使用 RepaintBoundary
    RepaintBoundary(
      child: ComplexWidget(),  // 隔离重绘
    )
    
    // 4. 使用 child 参数
    Consumer<Controller>(
      builder: (context, controller, child) {
        return Column(children: [child!, Text('${controller.value}')]);
      },
      child: ExpensiveWidget(),  // 不会重建
    )
    
    // 5. 避免不必要的 State
    // 使用 StatelessWidget 而不是 StatefulWidget
    // 只在需要状态时使用 StatefulWidget
    ''');
  }
}
```

### 2. 优化绘制

```dart
// 优化绘制
class OptimizePainting {
  void explain() {
    print('''
    优化绘制：
    
    // 1. 使用 CustomPaint
    CustomPaint(
      painter: MyPainter(),
      size: Size(200, 200),
    )
    
    // 2. 使用 Canvas
    class MyPainter extends CustomPainter {
      @override
      void paint(Canvas canvas, Size size) {
        final paint = Paint()
          ..color = Colors.blue
          ..style = PaintingStyle.fill;
        
        canvas.drawCircle(
          Offset(size.width / 2, size.height / 2),
          50,
          paint,
        );
      }
      
      @override
      bool shouldRepaint(covariant CustomPainter oldDelegate) => false;
    }
    
    // 3. 使用 Picture
    final recorder = PictureRecorder();
    final canvas = Canvas(recorder);
    // 绘制复杂图形
    final picture = recorder.endRecording();
    final image = picture.toImage(width, height);
    
    // 4. 使用 ImageFilter
    ImageFiltered(
      imageFilter: ImageFilter.blur(sigmaX: 5, sigmaY: 5),
      child: Container(...),
    )
    ''');
  }
}
```

## ⚠️ 注意事项

1. **目标帧率**：保持 60 FPS，至少 30 FPS
2. **慢帧阈值**：超过 16ms 的帧需要优化
3. **过度绘制**：避免不必要的绘制
4. **内存压力**：内存不足会影响帧率
5. **主线程**：避免阻塞主线程
