## 📖 布局基础

### 1. 布局概念

```dart
// 布局概念
class LayoutConcept {
  void explain() {
    print('''

    // - 子 Widget 返回尺寸
```

### 2. 约束系统

```dart
// 约束系统
class ConstraintSystem {
  void explain() {
    print('''
    约束系统：
    
    // 1. 什么是约束
    // - 父 Widget 传递给子 Widget 的尺寸限制
    // - 包括最小/最大宽度和高度
    // - 子 Widget 必须在约束范围内
    // - 约束向下传递
    
    // 2. 约束类型
    // - 宽松约束：没有最小限制
    // - 严格约束：有最小和最大限制
    // - 无限约束：没有最大限制
    // - 固定约束：固定尺寸
    
    // 3. 约束传递
    // - 父 Widget 传递约束给子 Widget
    // - 子 Widget 返回尺寸给父 Widget
    // - 父 Widget 确定子 Widget 位置
    // - 渲染到屏幕
    
    // 4. 约束示例
    // 使用 LayoutBuilder 查看约束
    LayoutBuilder(
      builder: (context, constraints) {
        print('约束: $constraints');
        return Container();
      },
    )
    ''');
  }
}
```

## 🔧 布局实现

### 1. 单子布局

```dart
// 单子布局
class SingleChildLayout {
  void explain() {
    print('''
    单子布局：
    
    // 1. 什么是单子布局
    // - 只有一个子 Widget
    // - 控制子 Widget 的位置和大小
    // - 提供装饰和约束
    
    // 2. 常用单子布局
    // - Container：容器
    // - Padding：内边距
    // - Align：对齐
    // - Center：居中
    // - SizedBox：固定尺寸
    
    // 3. Container 示例
    Container(
      width: 200,
      height: 100,
      padding: EdgeInsets.all(8),
      margin: EdgeInsets.all(8),
      decoration: BoxDecoration(
        color: Colors.blue,
        borderRadius: BorderRadius.circular(8),
      ),
      child: Text('Hello'),
    )
    
    // 4. 单子布局最佳实践
    // - 使用适当的 Widget
    // - 避免嵌套过深
    // - 使用 const 构造函数
    // - 考虑性能影响
    ''');
  }
}
```

### 2. 多子布局

```dart
// 多子布局
class MultiChildLayout {
  void explain() {
    print('''
    多子布局：
    
    // 1. 什么是多子布局
    // - 有多个子 Widget
    // - 控制子 Widget 的排列方式
    // - 提供布局方向
    
    // 2. 常用多子布局
    // - Row：水平布局
    // - Column：垂直布局
    // - Stack：层叠布局
    // - Flex：弹性布局
    // - Wrap：换行布局
    
    // 3. Row 示例
    Row(
      children: [
        Expanded(child: Text('Left')),
        Expanded(child: Text('Right')),
      ],
    )
    
    // 4. Column 示例
    Column(
      children: [
        Text('Top'),
        Text('Middle'),
        Text('Bottom'),
      ],
    )
    
    // 5. 多子布局最佳实践
    // - 使用适当的布局 Widget
    // - 避免嵌套过深
    // - 使用 const 构造函数
    // - 考虑性能影响
    ''');
  }
}
```
