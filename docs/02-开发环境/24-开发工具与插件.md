## 🛠️ 效率工具

### 1. 代码生成工具

```yaml
# pubspec.yaml
dev_dependencies:
  build_runner: ^2.4.0
  json_serializable: ^6.7.0
  freezed: ^2.4.0
  auto_route_generator: ^7.0.0
  retrofit_generator: ^7.0.0
```

```bash
# 运行代码生成
dart run build_runner build

# 监视文件变化
dart run build_runner watch

# 清除生成的文件
dart run build_runner clean
```

### 2. 代码格式化

```bash
# 格式化代码
dart format .

# 设置行长度
dart format --line-length 80 .

# 格式化特定文件
dart format lib/main.dart

# 检查格式
dart format --output=none --set-exit-if-changed .
```

### 3. 代码分析

```bash
# 分析代码
flutter analyze

# 自定义分析规则
# analysis_options.yaml
include: package:flutter_lints/flutter.yaml

analyzer:
  exclude:
    - "**/*.g.dart"
    - "**/*.freezed.dart"
  errors:
    invalid_annotation_target: ignore

linter:
  rules:
    - always_declare_return_types
    - always_use_package_imports
    - avoid_empty_else
    - avoid_print
    - prefer_const_constructors
    - prefer_const_declarations
    - use_key_in_widget_constructors
```

## 🐛 调试工具

### 1. Flutter Inspector

```
功能：
- 查看 Widget 树
- 查看布局约束
- 诊断布局问题
- 查看重绘区域

使用方法：
1. 运行应用（调试模式）
2. 打开 DevTools
3. 选择 Flutter Inspector 标签
4. 选择要检查的 Widget
```

### 2. DevTools

```bash
# 安装 DevTools
dart pub global activate devtools

# 运行 DevTools
dart devtools

# 在应用中使用
flutter run --debug
# 打开浏览器中显示的 URL

# DevTools 功能：
# - Inspector: Widget 树检查
# - Performance: 性能分析
# - CPU Profiler: CPU 分析
# - Memory: 内存分析
# - Debugger: 调试器
# - Network: 网络请求
# - Logging: 日志查看
```

### 3. 日志调试

```dart
// 使用 debugPrint（限制输出长度）
debugPrint('长文本...');

// 使用 developer.log
import 'dart:developer';
log('消息', name: 'my.app');

// 条件日志
void logDebug(String message) {
  if (kDebugMode) {
    print('[DEBUG] $message');
  }
}

// 格式化日志
class Logger {
  static void d(String message) {
    if (kDebugMode) print('[DEBUG] $message');
  }
  
  static void i(String message) {
    if (kDebugMode) print('[INFO] $message');
  }
  
  static void e(String message) {
    if (kDebugMode) print('[ERROR] $message');
  }
}
```

## 🚀 效率技巧

### 1. 快捷键

```
VS Code:
- Ctrl+Shift+P: 命令面板
- Ctrl+P: 快速打开文件
- F5: 开始调试
- Ctrl+S: 保存并热重载
- Ctrl+Shift+F: 全局搜索

Android Studio:
- Ctrl+Shift+A: 查找操作
- Shift+F10: 运行
- Shift+F9: 调试
- Ctrl+Alt+L: 格式化代码
- Ctrl+/: 切换注释
```

### 2. 代码片段

```json
// VS Code 代码片段
{
  "Stateful Widget": {
    "prefix": "stful",
    "body": [
      "class ${1:Name} extends StatefulWidget {",
      "  const ${1:Name}({Key? key}) : super(key: key);",
      "",
      "  @override",
      "  State<${1:Name}> createState() => _${1:Name}State();",
      "}",
      "",
      "class _${1:Name}State extends State<${1:Name}> {",
      "  @override",
      "  Widget build(BuildContext context) {",
      "    return Container();",
      "  }",
      "}"
    ]
  }
}
```

### 3. Git 工作流

```bash

# 分支管理

git checkout main

git push origin main
```

## 📝 常见问题

### 1. 依赖冲突

```
问题：版本冲突
解决：
1. flutter pub deps --no-dev
2. 查看依赖树
3. 手动指定版本
4. 使用 dependency_overrides
```

### 2. 缓存问题

```
问题：缓存导致的问题
解决：
flutter clean
flutter pub get
cd ios && pod install && cd ..
```

### 3. 构建问题

```
问题：构建失败
解决：
1. 检查 Flutter 版本
2. 更新依赖
3. 清除缓存
4. 检查平台配置
```
