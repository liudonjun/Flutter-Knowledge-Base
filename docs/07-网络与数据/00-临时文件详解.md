### 2. 临时文件实现

```dart
// 临时文件实现
class TemporaryFilesImplementation {
  void explain() {
    print('''
    临时文件实现：
    
    // 1. 获取临时目录
    final tempDir = await getTemporaryDirectory();
    final tempPath = tempDir.path;
    
    // 2. 创建临时文件
    final tempFile = File('$tempPath/temp_file.txt');
    await tempFile.writeAsString('Temporary data');
    
    // 3. 使用临时文件
    final content = await tempFile.readAsString();
    print(content);
    
    // 4. 清理临时文件
    await tempFile.delete();
    
    // 5. 临时文件最佳实践
    // - 及时清理
    // - 控制大小
    // - 错误处理
    // - 安全考虑
    
    // 6. 临时目录信息
    // - iOS: /tmp/
    // - Android: /data/data/package/cache/
    // - macOS: /var/folders/
    // - Windows: C:\Users\User\AppData\Local\Temp\
    ''');
  }
}
```
