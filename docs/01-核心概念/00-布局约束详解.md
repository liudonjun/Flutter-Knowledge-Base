# 布局约束

> 深入理解 Flutter 中的布局约束系统。

## 📖 布局约束基础

### 1. 什么是布局约束

```dart
// 布局约束概念
class LayoutConstraintsConcept {
  void explain() {
    print('''
    布局约束概念：
    
    // 1. 什么是布局约束
    // - 父 Widget 传递给子 Widget 的尺寸限制
    // - 包括最小/最大宽度和高度
    // - 子 Widget 必须在约束范围内
    // - 约束向下传递
    
    // 2. 布局约束的特点
    // - 约束传递：从父到子
    // - 尺寸返回：从子到父
    // - 位置确定：父级确定
    // - 约束解决：子级必须满足
    
    // 3. 布局约束的使用场景
    // - 布局计算：计算 Widget 大小
    // - 响应式设计：适应不同屏幕
    // - 约束解决：解决布局冲突
    // - 性能优化：优化布局计算
    ''');
  }
}
```

## 🔧 布局约束实现

### 1. 约束传递

```dart
// 约束传递
class ConstraintPassing {
  void explain() {
    print('''
    约束传递：
    
    // 1. 约束传递规则
    // - 父 Widget 传递约束给子 Widget
    // - 子 Widget 返回尺寸给父 Widget
    // - 父 Widget 确定子 Widget 位置
    // - 子 Widget 必须在约束范围内
    
    // 2. 约束传递示例
    class MyWidget extends StatelessWidget {
      @override
      Widget build(BuildContext context) {
        return Container(
          constraints: BoxConstraints(
            minWidth: 100,
            maxWidth: 200,
            minHeight: 100,
            maxHeight: 200,
          ),
          child: ChildWidget(),
        );
      }
    }
    
    class ChildWidget extends StatelessWidget {
      @override
      Widget build(BuildContext context) {
        return LayoutBuilder(
          builder: (context, constraints) {
            // 获取父 Widget 传递的约束
            final width = constraints.constrainWidth(150);
            final height = constraints.constrainHeight(150);
            
            return Container(
              width: width,
              height: height,
              color: Colors.blue,
            );
          },
        );
      }
    }
    
    // 3. 约束传递最佳实践
    // - 理解约束传递方向
    // - 使用合适的约束
    // - 考虑不同屏幕尺寸
    // - 测试不同设备
    ''');
  }
}
```

### 2. 约束解决

// 约束解决
class ConstraintResolution {

    约束解决：

    // - 子 Widget 必须在约束范围内
    // - 子 Widget 可以小于约束
    // - 子 Widget 不能大于约束
    // - 约束解决后确定尺寸
    
    // 2. 约束解决示例
    class MyWidget extends StatelessWidget {
      @override
      Widget build(BuildContext context) {

    class ChildWidget extends StatelessWidget {
      @override
      Widget build(BuildContext context) {
        return LayoutBuilder(
          builder: (context, constraints) {
            // 解决约束

            return Container(
              width: width,
              height: height,
              color: Colors.blue,
            );

        );

    }
    
    // 3. 约束解决最佳实践

    ''');
  }
}
