# 层叠布局

> 深入理解 Flutter 中的层叠布局。

## 📖 层叠布局基础

### 1. 什么是层叠布局

```dart
// 层叠布局概念
class StackLayoutConcept {
  void explain() {
    print('''
    层叠布局概念：
    
    // 1. 什么是层叠布局
    // - 子 Widget 可以重叠显示
    // - 通过定位控制位置
    // - 支持多种对齐方式
    // - 适合复杂的 UI 布局
    
    // 2. 层叠布局的特点
    // - 重叠显示：子 Widget 可以重叠
    // - 定位控制：通过 Positioned 控制位置
    // - 对齐方式：通过 alignment 控制对齐
    // - 尺寸控制：通过 fit 控制尺寸
    
    // 3. 层叠布局的使用场景
    // - 图片叠加：多张图片叠加
    // - 浮动按钮：悬浮在内容上方
    // - 标签叠加：标签叠加在图片上
    // - 复杂 UI：复杂的 UI 布局
    ''');
  }
}
```

### 2. Stack Widget

```dart
// Stack Widget
class StackWidget {
  void explain() {
    print('''
    Stack Widget：
    
    // 1. Stack 基本用法
    Stack(
      children: [
        Container(
          width: 200,
          height: 200,
          color: Colors.red,
        ),
        Container(
          width: 150,
          height: 150,
          color: Colors.green,
        ),
        Container(
          width: 100,
          height: 100,
          color: Colors.blue,
        ),
      ],
    )
    
    // 2. Stack 属性
    Stack(
      alignment: Alignment.center,  // 对齐方式
      fit: StackFit.loose,          // 尺寸控制
      clipBehavior: Clip.hardEdge,  // 裁剪行为
      textDirection: TextDirection.ltr,  // 文本方向
      children: [
        // 子 Widget
      ],
    )
    
    // 3. Positioned Widget
    Stack(
      children: [
        Positioned(
          left: 20,
          top: 20,
          child: Container(
            width: 100,
            height: 100,
            color: Colors.red,
          ),
        ),
        Positioned(
          right: 20,
          bottom: 20,
          child: Container(
            width: 100,
            height: 100,
            color: Colors.blue,
          ),
        ),
      ],
    )
    
    // 4. Stack 最佳实践
    // - 使用 Positioned 控制位置
    // - 使用 alignment 控制对齐
    // - 使用 fit 控制尺寸
    // - 使用 clipBehavior 控制裁剪
    ''');
  }
}
```

## 🔧 层叠布局实现

### 1. 对齐方式

```dart
// 对齐方式
class AlignmentMethods {
  void explain() {
    print('''
    对齐方式：
    
    // 1. Alignment 枚举
    // - Alignment.topLeft：左上角
    // - Alignment.topCenter：顶部居中
    // - Alignment.topRight：右上角
    // - Alignment.centerLeft：左侧居中
    // - Alignment.center：居中
    // - Alignment.centerRight：右侧居中
    // - Alignment.bottomLeft：左下角
    // - Alignment.bottomCenter：底部居中
    // - Alignment.bottomRight：右下角
    
    // 2. Alignment 示例
    Stack(
      alignment: Alignment.topLeft,
      children: [
        Container(width: 100, height: 100, color: Colors.red),
        Container(width: 50, height: 50, color: Colors.blue),
      ],
    )
    
    // 3. 自定义 Alignment
    Stack(
      alignment: Alignment(0.5, -0.5),  // 自定义对齐
      children: [
        Container(width: 100, height: 100, color: Colors.red),
        Container(width: 50, height: 50, color: Colors.blue),
      ],
    )
    
    // 4. 对齐方式最佳实践
    // - 使用预定义的 Alignment
    // - 使用自定义 Alignment
    // - 考虑文本方向
    // - 测试不同屏幕尺寸
    ''');
  }
}
```
