# 弹性布局

> 深入理解 Flutter 中的弹性布局。

## 📖 弹性布局基础

### 1. 什么是弹性布局

```dart
// 弹性布局概念
class FlexLayoutConcept {
  void explain() {
    print('''
    弹性布局概念：
    
    // 1. 什么是弹性布局
    // - 基于 Flex 的布局方式

    // - 支持弹性尺寸分配
    // - 支持多种对齐方式
    
    // 2. 弹性布局的特点

    // - 弹性尺寸：按比例分配空间

    // - 间距控制：控制子 Widget 间距
    
    // 3. 弹性布局的使用场景
    // - 水平排列：水平排列子 Widget
    // - 垂直排列：垂直排列子 Widget
    // - 弹性分配：按比例分配空间
    // - 复杂布局：复杂的 UI 布局
    ''');
  }
}
```

### 2. Flex Widget

```dart
// Flex Widget
class FlexWidget {
  void explain() {
    print('''
    Flex Widget：
    
    // 1. Flex 基本用法
    Flex(
      direction: Axis.horizontal,
      children: [
        Expanded(
          flex: 1,
          child: Container(color: Colors.red),
        ),
        Expanded(
          flex: 2,
          child: Container(color: Colors.blue),
        ),
      ],
    )
    
    // 2. Flex 属性
    Flex(
      direction: Axis.horizontal,  // 方向
      mainAxisAlignment: MainAxisAlignment.start,  // 主轴对齐
      crossAxisAlignment: CrossAxisAlignment.center,  // 交叉轴对齐
      mainAxisSize: MainAxisSize.max,  // 主轴尺寸
      textDirection: TextDirection.ltr,  // 文本方向
      verticalDirection: VerticalDirection.down,  // 垂直方向
      children: [
        // 子 Widget
      ],
    )
    
    // 3. Row 和 Column
    // Row 是水平方向的 Flex
    Row(
      children: [
        Expanded(child: Container(color: Colors.red)),
        Expanded(child: Container(color: Colors.blue)),
      ],
    )
    
    // Column 是垂直方向的 Flex
    Column(
      children: [
        Expanded(child: Container(color: Colors.red)),
        Expanded(child: Container(color: Colors.blue)),
      ],
    )
    
    // 4. Flex 最佳实践
    // - 使用 Row 或 Column
    // - 使用 Expanded 控制弹性
    // - 使用 Spacer 控制间距
    // - 使用对齐方式控制位置
    ''');
  }
}
```

## 🔧 弹性布局实现

### 1. Expanded Widget

```dart
// Expanded Widget
class ExpandedWidget {
  void explain() {
    print('''
    Expanded Widget：
    
    // 1. Expanded 基本用法
    Row(
      children: [
        Expanded(
          flex: 1,
          child: Container(color: Colors.red),
        ),
        Expanded(
          flex: 2,
          child: Container(color: Colors.blue),
        ),
      ],
    )
    
    // 2. Expanded 属性
    Expanded(
      flex: 2,  // 弹性比例
      child: Container(color: Colors.red),
    )
    
    // 3. Flexible Widget
    Flexible(
      flex: 1,
      fit: FlexFit.loose,  // 适配方式
      child: Container(color: Colors.red),
    )
    
    // 4. Expanded 最佳实践
    // - 使用 flex 控制比例
    // - 使用 Flexible 控制适配
    // - 考虑不同屏幕尺寸
    // - 测试不同设备
    ''');
  }
}
```

### 2. 对齐方式

```dart
// 对齐方式
class AlignmentMethods {
  void explain() {
    print('''
    对齐方式：
    
    // 1. MainAxisAlignment 枚举
    // - MainAxisAlignment.start：起始位置
    // - MainAxisAlignment.end：结束位置
    // - MainAxisAlignment.center：居中
    // - MainAxisAlignment.spaceBetween：间距相等
    // - MainAxisAlignment.spaceAround：周围间距相等
    // - MainAxisAlignment.spaceEvenly：完全相等
    
    // 2. CrossAxisAlignment 枚举
    // - CrossAxisAlignment.start：起始位置
    // - CrossAxisAlignment.end：结束位置
    // - CrossAxisAlignment.center：居中
    // - CrossAxisAlignment.stretch：拉伸
    // - CrossAxisAlignment.baseline：基线对齐
    
    // 3. 对齐方式示例
    Row(
      mainAxisAlignment: MainAxisAlignment.spaceEvenly,
      crossAxisAlignment: CrossAxisAlignment.center,
      children: [
        Container(width: 50, height: 50, color: Colors.red),
        Container(width: 50, height: 50, color: Colors.blue),
        Container(width: 50, height: 50, color: Colors.green),
      ],
    )
    
    // 4. 对齐方式最佳实践
    // - 使用合适的对齐方式
    // - 考虑不同屏幕尺寸
    // - 测试不同设备
    // - 保持 UI 一致性
    ''');
  }
}
```
