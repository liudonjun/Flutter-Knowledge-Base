# 侧边栏导航详解

> 深入理解 Flutter 中的侧边栏导航。

## 📖 侧边栏导航基础

### 1. 什么是侧边栏导航

```dart
// 侧边栏导航概念
class SidebarNavigationConcept {
  void explain() {
    print('''
    侧边栏导航概念：
    
    // 1. 什么是侧边栏导航
    // - 从屏幕边缘滑出的导航面板
    // - 包含应用的主要导航链接
    // - 支持用户信息显示
    // - 遵循 Material Design
    
    // 2. 侧边栏导航的特点
    // - 节省空间：不占用主屏幕空间
    // - 快速访问：可以快速切换页面
    // - 用户信息：可以显示用户信息
    // - 可定制：支持自定义样式
    
    // 3. 侧边栏导航的使用场景
    // - 主要导航：应用主导航
    // - 用户账户：显示用户信息
    // - 设置页面：快速访问设置
    // - 帮助支持：提供帮助信息
    ''');
  }
}
```

### 1. 菜单项

```dart
// 菜单项
class MenuItems {

    print('''
```

### 2. 高级功能

```dart
// 高级功能
class AdvancedFeatures {
  void explain() {
    print('''
    高级功能：
    
    // 1. 嵌套导航
    Navigator(
      key: _navigatorKey,
      onGenerateRoute: (settings) {
        return MaterialPageRoute(
          builder: (context) => getPage(settings.name),
        );
      },
    )
    
    // 2. 状态管理
    class SidebarState {
      static bool isOpen = false;
      
      static void toggle() {
        isOpen = !isOpen;
      }
    }
    
    // 3. 动画效果
    Drawer(
      elevation: 16,
      child: Container(
        decoration: BoxDecoration(
          gradient: LinearGradient(
            begin: Alignment.topLeft,
            end: Alignment.bottomRight,
            colors: [Colors.blue, Colors.purple],
          ),
        ),
        child: ListView(
          children: [
            // ... 菜单项
          ],
        ),
      ),
    )
    
    // 4. 响应式设计
    LayoutBuilder(
      builder: (context, constraints) {
        if (constraints.maxWidth > 600) {
          return NavigationRail(
            // 宽屏使用 NavigationRail
          );
        } else {
          return Drawer(
            // 窄屏使用 Drawer
          );
        }
      },
    )
    ''');
  }
}
```
