# 动态路由详解

> 深入理解 Flutter 中的动态路由。

## 📖 动态路由基础

### 1. 什么是动态路由

```dart
// 动态路由概念
class DynamicRoutingConcept {
  void explain() {
    print('''
    动态路由概念：
    
    // 1. 什么是动态路由
    // - 运行时动态生成的路由
    // - 基于参数动态匹配
    // - 支持复杂路由逻辑
    // - 提高路由灵活性
    
    // 2. 动态路由的特点
    // - 灵活性：可以根据参数动态生成
    // - 可扩展：支持复杂路由逻辑
    // - 类型安全：编译时检查
    // - 易于维护：路由逻辑集中
    
    // 3. 动态路由的使用场景
    // - 详情页面：根据 ID 显示详情
    // - 用户页面：根据用户名显示页面
    // - 搜索页面：根据搜索词显示结果
    // - 分类页面：根据分类显示内容
    ''');
  }
}
```

### 2. 动态路由配置

```dart
// 动态路由配置
class DynamicRoutingConfiguration {
  void explain() {
    print('''
    动态路由配置：
    
    // 1. 使用 onGenerateRoute
    MaterialApp(
      onGenerateRoute: (settings) {
        final uri = Uri.parse(settings.name!);
        
        if (uri.pathSegments.length == 2 && uri.pathSegments[0] == 'user') {
          final username = uri.pathSegments[1];
          return MaterialPageRoute(
            builder: (context) => UserPage(username: username),
          );
        }
        
        return MaterialPageRoute(
          builder: (context) => NotFoundPage(),
        );
      },
    )
    
    // 2. 使用路由表
    class AppRoutes {
      static const String HOME = '/';
      static const String USER = '/user/:username';
      static const String PRODUCT = '/product/:id';
    }
    
    // 3. 使用 GetX
    final routes = [
      GetPage(
        name: AppRoutes.USER,
        page: () {
          final username = Get.parameters['username']!;
          return UserPage(username: username);
        },
      ),
    ];
    ''');
  }
}
```

## 🔧 动态路由使用

### 1. 路径参数

```dart
// 路径参数
class PathParameters {
  void explain() {
    print('''
    路径参数：
    
    // 1. 定义路径参数
    // /user/:username
    // /product/:id
    // /category/:slug
    
    // 2. 获取路径参数
    final uri = Uri.parse(settings.name!);
    
    if (uri.pathSegments.length == 2 && uri.pathSegments[0] == 'user') {
      final username = uri.pathSegments[1];
      // 使用 username
    }
    
    // 3. 使用 GetX 获取参数
    final username = Get.parameters['username'];
    final id = Get.parameters['id'];
    
    // 4. 参数验证
    bool validateParameters(Map<String, String> params) {
      if (params.containsKey('id')) {
        final id = int.tryParse(params['id']!);
        return id != null && id > 0;
      }
      return false;
    }
    ''');
  }
}
```

### 2. 查询参数

```dart
// 查询参数
class QueryParameters {
  void explain() {
    print('''
    查询参数：
    
    // 1. 定义查询参数
    // /search?query=flutter
    // /products?category=books&sort=price
    
    // 2. 获取查询参数
    final uri = Uri.parse(settings.name!);
    final query = uri.queryParameters['query'];
    final category = uri.queryParameters['category'];
    final sort = uri.queryParameters['sort'];
    
    // 3. 使用 GetX 获取查询参数
    final query = Get.parameters['query'];
    final category = Get.parameters['category'];
    
    // 4. 查询参数验证
    bool validateQueryParameters(Map<String, String> params) {
      if (params.containsKey('query')) {
        final query = params['query']!;
        return query.isNotEmpty && query.length <= 100;
      }
      return false;
    }
    ''');
  }
}
```
