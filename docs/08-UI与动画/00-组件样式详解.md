> 深入理解 Flutter 中的组件样式。

## 📖 组件样式基础

### 1. 什么是组件样式

```dart
// 组件样式概念
class ComponentStylesConcept {
  void explain() {
    print('''
    组件样式概念：

    // - 组件的外观样式
    // - 统一的视觉风格
    // - 支持自定义样式
    // - 支持主题样式
    
    // 2. 组件样式的特点
    // - 统一风格：统一的视觉风格
    // - 可定制：支持自定义样式
    // - 主题支持：支持主题样式

    // 3. 组件样式的类型
    // - 内置样式：组件默认样式
    // - 主题样式：主题中定义的样式
    // - 自定义样式：自定义样式
```

### 2. 组件样式实现

```dart
// 组件样式实现
class ComponentStylesImplementation {
  void explain() {
    print('''
    组件样式实现：
    
    // 1. 使用主题样式
    ThemeData(
      elevatedButtonTheme: ElevatedButtonThemeData(
        style: ElevatedButton.styleFrom(
          primary: Colors.blue,
          onPrimary: Colors.white,
          padding: EdgeInsets.symmetric(horizontal: 16, vertical: 8),
        ),
      ),
      textTheme: TextTheme(
        button: TextStyle(
          fontSize: 16,
          fontWeight: FontWeight.bold,
        ),
      ),
    )
    
    // 2. 使用自定义样式
    Container(
      decoration: BoxDecoration(
        color: Colors.blue,
        borderRadius: BorderRadius.circular(8),
        boxShadow: [
          BoxShadow(
            color: Colors.black12,
            blurRadius: 4,
            offset: Offset(0, 2),
          ),
        ],
      ),
      child: Text('Hello'),
    )
    
    // 3. 使用样式类
    class AppStyles {
      static const buttonStyle = TextStyle(
        fontSize: 16,
        fontWeight: FontWeight.bold,
        color: Colors.white,
      );
      
      static const cardDecoration = BoxDecoration(
        color: Colors.white,
        borderRadius: BorderRadius.circular(8),
        boxShadow: [
          BoxShadow(
            color: Colors.black12,
            blurRadius: 4,
          ),
        ],
      );
    }
    
    // 4. 组件样式最佳实践
    // - 使用主题样式
    // - 保持一致性
    // - 考虑可访问性
    // - 支持动态样式
    ''');
  }
}
```
