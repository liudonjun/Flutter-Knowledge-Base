# 隐式动画详解

> 深入理解 Flutter 中的隐式动画。

## 📖 隐式动画基础

### 1. 什么是隐式动画

```dart
// 隐式动画概念
class ImplicitAnimationConcept {
  void explain() {
    print('''
    隐式动画概念：
    
    // 1. 什么是隐式动画
    // - 自动动画：属性变化时自动动画
    // - 无需手动控制
    // - 简化动画实现
    // - 提高开发效率
    
    // 2. 隐式动画的特点
    // - 自动化：自动处理动画
    // - 简单易用：API 简洁
    // - 性能优化：性能优化
    // - 可定制：支持自定义
    
    // 3. 隐式动画的类型
    // - AnimatedContainer：容器动画
    // - AnimatedOpacity：透明度动画
    // - AnimatedPositioned：位置动画
    // - AnimatedDefaultTextStyle：文本动画
    
    // 4. 隐式动画的使用场景
    // - 按钮效果
    // - 列表动画
    // - 状态变化
    // - 交互反馈
    ''');
  }
}
```

### 2. 隐式动画实现

```dart
// 隐式动画实现
class ImplicitAnimationImplementation {
  void explain() {
    print('''
    隐式动画实现：
    
    // 1. AnimatedContainer
    AnimatedContainer(
      duration: Duration(seconds: 1),
      width: _isExpanded ? 200 : 100,
      height: _isExpanded ? 200 : 100,
      color: _isExpanded ? Colors.blue : Colors.red,
      curve: Curves.easeInOut,
      child: Text('Hello'),
    )
    
    // 2. AnimatedOpacity
    AnimatedOpacity(
      duration: Duration(seconds: 1),
      opacity: _isVisible ? 1.0 : 0.0,
      child: Text('Hello'),
    )
    
    // 3. AnimatedPositioned
    Stack(
      children: [
        AnimatedPositioned(
          duration: Duration(seconds: 1),
          left: _position.dx,
          top: _position.dy,
          child: Container(
            width: 50,
            height: 50,
            color: Colors.blue,
          ),
        ),
      ],
    )
    
    // 4. AnimatedDefaultTextStyle
    AnimatedDefaultTextStyle(
      duration: Duration(seconds: 1),
      style: TextStyle(
        fontSize: _fontSize,
        color: _fontColor,
      ),
      child: Text('Hello'),
    )
    
    // 5. 隐式动画最佳实践
    // - 使用适当持续时间
    // - 使用适当曲线
    // - 考虑性能
    // - 测试动画效果
    ''');
  }
}
```
