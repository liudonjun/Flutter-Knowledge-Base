## 📖 物理动画基础

### 2. 物理动画实现

```dart
// 物理动画实现
class PhysicsAnimationImplementation {
  void explain() {
    print('''
    物理动画实现：
    
    // 1. 弹簧动画
    final simulation = SpringSimulation(
      SpringDescription.withDampingRatio(
        mass: 1.0,
        stiffness: 100.0,
        ratio: 0.5,
      ),
      0.0,  // 起始位置
      100.0,  // 结束位置
      0.0,  // 起始速度
    );
    
    // 2. 使用弹簧动画
    _controller.animateWith(simulation);
    
    // 3. 重力动画
    final simulation = GravitySimulation(
      9.8,  // 重力加速度
      0.0,  // 起始位置
      100.0,  // 结束位置
      0.0,  // 起始速度
    );
    
    // 4. 摩擦动画
    final simulation = FrictionSimulation(
      0.1,  // 摩擦系数
      0.0,  // 起始位置
      100.0,  // 起始速度
    );
    
    // 5. 物理动画最佳实践
    // - 使用适当物理参数
    // - 考虑性能
    // - 优化物理效果
    // - 测试不同设备
    ''');
  }
}
```
