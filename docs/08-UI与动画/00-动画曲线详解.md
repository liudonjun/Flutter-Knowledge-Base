### 2. 动画曲线实现

```dart
// 动画曲线实现
class AnimationCurvesImplementation {
  void explain() {
    print('''
    动画曲线实现：
    
    // 1. 使用内置曲线
    final animation = CurvedAnimation(
      parent: _controller,
      curve: Curves.easeInOut,
    );
    
    // 2. 常用曲线
    // Curves.linear：线性
    // Curves.ease：缓动
    // Curves.easeIn：缓入
    // Curves.easeOut：缓出
    // Curves.easeInOut：缓入缓出
    // Curves.bounceIn：弹入
    // Curves.bounceOut：弹出
    // Curves.elasticIn：弹性入
    // Curves.elasticOut：弹性出
    
    // 3. 自定义曲线
    final customCurve = Cubic(0.68, -0.55, 0.27, 1.55);
    final animation = CurvedAnimation(
      parent: _controller,
      curve: customCurve,
    );
    
    // 4. 使用 TweenSequence
    final animation = TweenSequence<double>([
      TweenSequenceItem(
        tween: Tween<double>(begin: 0.0, end: 1.2),
        weight: 1.0,
      ),
      TweenSequenceItem(
        tween: Tween<double>(begin: 1.2, end: 1.0),
        weight: 1.0,
      ),
    ]).animate(_controller);
    
    // 5. 动画曲线最佳实践
    // - 选择适当曲线
    // - 考虑用户体验
    // - 测试动画效果
    // - 性能考虑
    ''');
  }
}
```
