# 颜色系统详解

> 深入理解 Flutter 中的颜色系统。

## 📖 颜色系统基础

### 1. 什么是颜色系统

```dart
// 颜色系统概念
class ColorSystemConcept {
  void explain() {
    print('''
    颜色系统概念：
    
    // 1. 什么是颜色系统
    // - 颜色的管理方式
    // - 统一的颜色定义
    // - 支持主题颜色
    // - 支持动态颜色
    
    // 2. 颜色系统的特点
    // - 统一管理：统一管理颜色
    // - 可定制：支持自定义颜色
    // - 动态切换：支持动态切换
    // - 一致性：保持颜色一致
    
    // 3. 颜色系统的类型
    // - 主题颜色：ThemeData 中的颜色
    // - 自定义颜色：自定义颜色值
    // - 材料颜色：Material 颜色
    // - 品牌颜色：品牌特定颜色
    
    // 4. 颜色系统的使用场景
    // - 主题配置
    // - 品牌定制
    // - 暗黑模式
    // - 动态主题
    ''');
  }
}
```

### 2. 颜色系统实现

```dart
// 颜色系统实现
class ColorSystemImplementation {
  void explain() {
    print('''
    颜色系统实现：
    
    // 1. 使用 Colors
    Container(
      color: Colors.blue,
      child: Text('Hello'),
    )
    
    // 2. 使用 Color
    Container(
      color: Color(0xFF2196F3),
      child: Text('Hello'),
    )
    
    // 3. 使用 ThemeData
    ThemeData(
      primaryColor: Colors.blue,
      accentColor: Colors.orange,
      backgroundColor: Colors.white,
      textTheme: TextTheme(
        bodyText1: TextStyle(color: Colors.black),
      ),
    )
    
    // 4. 自定义颜色系统
    class AppColors {
      static const Color primary = Color(0xFF2196F3);
      static const Color secondary = Color(0xFFFF9800);
      static const Color background = Color(0xFFFFFFFF);
      static const Color text = Color(0xFF000000);
    }
    
    // 5. 颜色系统最佳实践
    // - 使用主题颜色
    // - 保持一致性
    // - 考虑可访问性
    // - 支持暗黑模式
    ''');
  }
}
```
