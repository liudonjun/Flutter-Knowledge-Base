# 响应式设计详解

> 深入理解 Flutter 中的响应式设计。

## 📖 响应式设计基础

### 1. 什么是响应式设计

```dart
// 响应式设计概念
class ResponsiveDesignConcept {
  void explain() {
    print('''
    响应式设计概念：
    
    // 1. 什么是响应式设计
    // - 适配不同屏幕尺寸
    // - 提供一致的用户体验
    // - 支持多种设备
    // - 优化布局
    
    // 2. 响应式设计的特点
    // - 自适应：自适应屏幕尺寸
    // - 一致性：保持一致的体验
    // - 灵活性：灵活的布局
    // - 性能优化：性能优化
    
    // 3. 响应式设计的方式
    // - MediaQuery：媒体查询
    // - LayoutBuilder：布局构建器
    // - Flexible：弹性布局
    // - AspectRatio：宽高比
    
    // 4. 响应式设计的使用场景
    // - 手机和平板
    // - 横竖屏切换
    // - 不同分辨率
    // - 折叠屏设备
    ''');
  }
}
```

### 2. 响应式设计实现

```dart
// 响应式设计实现
class ResponsiveDesignImplementation {
  void explain() {
    print('''
    响应式设计实现：
    
    // 1. 使用 MediaQuery
    final screenWidth = MediaQuery.of(context).size.width;
    final screenHeight = MediaQuery.of(context).size.height;
    
    // 2. 使用 LayoutBuilder
    LayoutBuilder(
      builder: (context, constraints) {
        if (constraints.maxWidth > 600) {
          return WideLayout();
        } else {
          return NarrowLayout();
        }
      },
    )
    
    // 3. 使用 Flexible
    Row(
      children: [
        Flexible(
          flex: 1,
          child: Container(color: Colors.red),
        ),
        Flexible(
          flex: 2,
          child: Container(color: Colors.blue),
        ),
      ],
    )
    
    // 4. 使用 OrientationBuilder
    OrientationBuilder(
      builder: (context, orientation) {
        if (orientation == Orientation.landscape) {
          return LandscapeLayout();
        } else {
          return PortraitLayout();
        }
      },
    )
    
    // 5. 响应式设计最佳实践
    // - 使用 MediaQuery
    // - 使用 LayoutBuilder
    // - 测试不同设备
    // - 考虑性能
    ''');
  }
}
```
