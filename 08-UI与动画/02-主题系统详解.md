# Flutter 主题系统详解

> 掌握 Flutter 主题系统，创建一致的用户界面。

## 🎨 主题系统概述

### 1. 主题系统组成
```dart
// Flutter 主题系统
// 1. ThemeData - 主题数据
// 2. ColorScheme - 颜色方案
// 3. TextTheme - 文本主题
// 4. ButtonTheme - 按钮主题
// 5. AppBarTheme - 应用栏主题

// 主题类型
// 1. Material Design - Google 设计语言
// 2. Cupertino - iOS 设计语言
// 3. 自定义主题 - 企业设计系统
```

### 2. 主题应用方式
```dart
// 应用主题
MaterialApp(
  theme: ThemeData.light(), // 浅色主题
  darkTheme: ThemeData.dark(), // 深色主题
  themeMode: ThemeMode.system, // 跟随系统
  home: MyHomePage(),
)

// 获取当前主题
Theme.of(context) // 获取 ThemeData
Theme.of(context).colorScheme // 获取 ColorScheme
Theme.of(context).textTheme // 获取 TextTheme
```

### 2. 主题切换
```dart
// 主题切换服务
class ThemeService extends ChangeNotifier {
  ThemeMode _themeMode = ThemeMode.system;
  
  ThemeMode get themeMode => _themeMode;
  
  void setThemeMode(ThemeMode mode) {
    _themeMode = mode;
    notifyListeners();
  }
  
  void toggleTheme() {
    _themeMode = _themeMode == ThemeMode.light
        ? ThemeMode.dark
        : ThemeMode.light;
    notifyListeners();
  }
}

// 在应用中使用
class MyApp extends StatefulWidget {
  @override
  _MyAppState createState() => _MyAppState();
}

class _MyAppState extends State<MyApp> {
  final ThemeService _themeService = ThemeService();
  
  @override
  Widget build(BuildContext context) {
    return ChangeNotifierProvider.value(
      value: _themeService,
      child: Consumer<ThemeService>(
        builder: (context, themeService, child) {
          return MaterialApp(
            theme: ThemeData.light(),
            darkTheme: ThemeData.dark(),
            themeMode: themeService.themeMode,
            home: MyHomePage(),
          );
        },
      ),
    );
  }
}
```

```dart
class ResponsiveTheme {

    final screenWidth = MediaQuery.of(context).size.width;

      return ThemeData(
```

## 🎯 主题最佳实践

### 1. 主题设计原则
```dart
// 主题设计原则
// 1. 一致性
//    - 使用统一的颜色方案
//    - 使用统一的字体系统
//    - 使用统一的间距系统

// 2. 可访问性
//    - 确保足够的对比度
//    - 支持动态字体大小
//    - 支持屏幕阅读器

// 3. 灵活性
//    - 支持浅色/深色主题
//    - 支持动态颜色
//    - 支持自定义主题

// 4. 性能
//    - 避免过度复杂的主题
//    - 使用 const 主题
//    - 缓存主题数据
```

### 2. 主题管理
```dart
// 主题管理最佳实践
class ThemeManager {
  static final ThemeManager _instance = ThemeManager._internal();
  factory ThemeManager() => _instance;
  ThemeManager._internal();
  
  // 主题缓存
  final Map<String, ThemeData> _themeCache = {};
  
  // 获取主题
  ThemeData getTheme(String themeName) {
    if (!_themeCache.containsKey(themeName)) {
      _themeCache[themeName] = _createTheme(themeName);
    }
    return _themeCache[themeName]!;
  }
  
  // 创建主题
  ThemeData _createTheme(String themeName) {
    switch (themeName) {
      case 'light':
        return ThemeData.light();
      case 'dark':
        return ThemeData.dark();
      case 'blue':
        return ThemeData(
          primarySwatch: Colors.blue,
        );
      default:
        return ThemeData.light();
    }
  }
  
  // 清除缓存
  void clearCache() {
    _themeCache.clear();
  }
}
```
