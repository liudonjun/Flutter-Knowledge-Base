> 从零开始搭建 Flutter 开发环境，为开发做好准备。

## 📦 Flutter SDK 安装

### 1. 下载 Flutter SDK
```bash
# 方法一: 使用 Git 克隆 (推荐)
git clone https://github.com/flutter/flutter.git -b stable

# 方法二: 下载压缩包
# 访问 https://flutter.dev/docs/get-started/install
# 下载适合你操作系统的版本

# 方法三: 使用包管理器 (macOS)
# 使用 Homebrew
brew install --cask flutter

# 使用 MacPorts
sudo port install flutter
```

### 2. 配置环境变量
```bash
# macOS/Linux
# 编辑 ~/.bashrc 或 ~/.zshrc
export PATH="$PATH:`pwd`/flutter/bin"

# 或者添加到 PATH
export PATH="$PATH:/path/to/flutter/bin"

# 使配置生效
source ~/.bashrc  # 或 source ~/.zshrc

# Windows
# 1. 右键"此电脑" -> 属性 -> 高级系统设置 -> 环境变量
# 2. 在 Path 中添加 Flutter SDK 的 bin 目录路径
# 3. 重启命令行

# 验证安装
flutter --version
```

### 3. 配置国内镜像 (可选)
```bash
# 使用国内镜像加速下载
export PUB_HOSTED_URL=https://pub.flutter-io.cn
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn

# 永久配置 (添加到 ~/.bashrc 或 ~/.zshrc)
echo 'export PUB_HOSTED_URL=https://pub.flutter-io.cn' >> ~/.bashrc
echo 'export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn' >> ~/.bashrc
source ~/.bashrc
```

## 🔧 开发工具安装

### 1. Android Studio
```bash
# 下载 Android Studio
# 访问 https://developer.android.com/studio

# 安装步骤
# 1. 下载并安装 Android Studio
# 2. 启动 Android Studio
# 3. 进入 Configure -> Plugins
# 4. 搜索并安装 Flutter 和 Dart 插件
# 5. 重启 Android Studio

# 配置 Android SDK
# 1. 进入 Configure -> SDK Manager
# 2. 选择 SDK Platforms 标签
# 3. 勾选 Android 13.0 (Tiramisu) 或其他版本
# 4. 选择 SDK Tools 标签
# 5. 勾选 Android SDK Command-line Tools
# 6. 点击 Apply 安装

# 配置 Android 模拟器
# 1. 进入 Configure -> AVD Manager
# 2. 点击 Create Virtual Device
# 3. 选择设备型号
# 4. 选择系统镜像
# 5. 配置模拟器设置
# 6. 启动模拟器
```

### 2. VS Code
```bash
# 下载 VS Code
# 访问 https://code.visualstudio.com/

# 安装 Flutter 和 Dart 扩展
# 1. 启动 VS Code
# 2. 打开扩展视图 (Ctrl+Shift+X)
# 3. 搜索 Flutter 并安装
# 4. 搜索 Dart 并安装
# 5. 重启 VS Code

# 配置 VS Code 设置
# 1. 打开设置 (Ctrl+,)
# 2. 搜索 flutter.sdkPath
# 3. 设置 Flutter SDK 路径
# 4. 搜索 dart.sdkPath
# 5. 设置 Dart SDK 路径 (通常在 flutter/bin/cache/dart-sdk)
```

### 3. IntelliJ IDEA
```bash
# 下载 IntelliJ IDEA
# 访问 https://www.jetbrains.com/idea/

# 安装 Flutter 和 Dart 插件
# 1. 启动 IntelliJ IDEA
# 2. 进入 File -> Settings -> Plugins
# 3. 搜索 Flutter 并安装
# 4. 搜索 Dart 并安装
# 5. 重启 IntelliJ IDEA

# 配置 Flutter SDK
# 1. 进入 File -> Settings -> Languages & Frameworks -> Flutter
# 2. 设置 Flutter SDK 路径
# 3. 点击 Apply
```

## 📱 移动设备配置

### 1. Android 真机调试
```bash
# 启用开发者选项
# 1. 进入设置 -> 关于手机
# 2. 连续点击"版本号" 7 次
# 3. 返回设置 -> 系统 -> 开发者选项

# 启用 USB 调试
# 1. 进入开发者选项
# 2. 启用 USB 调试
# 3. 连接手机到电脑
# 4. 在手机上允许 USB 调试

# 验证连接
flutter devices
# 应该能看到你的设备

# 运行应用
flutter run
```

### 2. iOS 真机调试 (仅 macOS)
```bash
# 配置 Xcode
# 1. 安装 Xcode (从 App Store)
# 2. 启动 Xcode 并同意许可协议
# 3. 安装 Xcode 命令行工具
xcode-select --install

# 配置 iOS 模拟器
# 1. 打开 Xcode
# 2. 进入 Xcode -> Preferences -> Components
# 3. 下载 iOS 模拟器

# 启用开发者模式
# 1. 连接 iPhone 到 Mac
# 2. 在 iPhone 上信任电脑
# 3. 在 Xcode 中添加开发者账户
# 4. 在 iPhone 上启用开发者模式

# 验证连接
flutter devices
# 应该能看到你的 iOS 设备

# 运行应用
flutter run
```

## 🛠️ 常见问题解决

### 1. 网络问题
```bash
# 问题: 下载速度慢或失败
# 解决方案:

# 1. 使用国内镜像
export PUB_HOSTED_URL=https://pub.flutter-io.cn
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn

# 2. 配置代理
export http_proxy=http://proxy.example.com:8080
export https_proxy=http://proxy.example.com:8080

# 3. 使用 VPN
# 配置 VPN 连接

# 4. 手动下载 SDK
# 从 Flutter 官网下载离线包
```

### 2. 权限问题
```bash
# 问题: 权限不足
# 解决方案:

# macOS
# 1. 修复 Xcode 权限
sudo xcode-select --switch /Applications/Xcode.app/Contents/Developer
sudo xcodebuild -runFirstLaunch

# 2. 修复 Android SDK 权限
sudo chown -R $(whoami) ~/Library/Android/sdk

# Linux
# 1. 添加用户到 dialout 组
sudo usermod -a -G dialout $USER

# 2. 重启系统
sudo reboot

# Windows
# 1. 以管理员身份运行命令行
# 2. 检查防病毒软件设置
```

### 3. 设备连接问题
```bash
# 问题: 设备无法识别
# 解决方案:

# Android
# 1. 检查 USB 调试是否启用
# 2. 更换 USB 数据线
# 3. 更换 USB 端口
# 4. 重启 ADB 服务
adb kill-server
adb start-server

# iOS
# 1. 检查是否信任电脑
# 2. 检查开发者证书
# 3. 重启 Xcode
# 4. 重启设备

# 通用解决方案
# 1. 重启电脑
# 2. 更新驱动程序
# 3. 检查防火墙设置
```

## 🎯 最佳实践

### 1. 环境管理
```bash
# 使用 FVM 管理 Flutter 版本
# 安装 FVM
dart pub global activate fvm

# 安装 Flutter 版本
fvm install stable
fvm install beta

# 使用特定版本
fvm use stable

# 全局使用
fvm global stable
```

### 2. 项目配置
```yaml
# pubspec.yaml 示例
name: my_app
description: A new Flutter project

# 版本管理
version: 1.0.0+1

# 环境配置
environment:
  sdk: ">=2.17.0 <3.0.0"
  flutter: ">=3.0.0"

# 依赖管理
dependencies:
  flutter:
    sdk: flutter
  cupertino_icons: ^1.0.2
  provider: ^6.0.0
  dio: ^5.0.0

# 开发依赖
dev_dependencies:
  flutter_test:
    sdk: flutter
  flutter_lints: ^2.0.0

# Flutter 配置
flutter:
  uses-material-design: true
  
  # 资源配置
  assets:
    - assets/images/
    - assets/fonts/
  
  # 字体配置
  fonts:
    - family: CustomFont
      fonts:
        - asset: assets/fonts/CustomFont-Regular.ttf
        - asset: assets/fonts/CustomFont-Bold.ttf
          weight: 700
```
