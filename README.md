# Flutter 知识库

> 系统化的 Flutter 开发学习资源，从入门到精通的完整学习路径。

## 📚 知识库概览

这是一个基于 Obsidian 和 VitePress 构建的 Flutter 学习知识库，包含：

- **20+ 学习模块** - 覆盖 Flutter 全栈开发
- **100+ 代码示例** - 实用的 Dart 和 Flutter 代码
- **完整学习路径** - 从初学者到高级开发者
- **可视化图表** - 架构图、流程图、概念图

## 🚀 快速开始

### 本地运行

```bash
# 克隆仓库
git clone git@github.com:liudonjun/Flutter-Knowledge-Base.git
cd Flutter-Knowledge-Base

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问 http://localhost:5173/
```

### 在线访问

- **GitHub Pages**: https://liudonjun.github.io/Flutter-Knowledge-Base/
- **Obsidian**: 直接打开 `docs/` 目录
